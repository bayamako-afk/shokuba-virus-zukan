# Season 1前半 実写版用キャラ基準資産参照ルール

## 1. この文書の目的

本書は、**Season 1前半（EP001〜EP005）の実写版において、EP001で整備した基準画像を共通参照資産として継続利用するための運用基準**を定める。実写版はアニメ版よりも顔立ち、髪型、年齢感、服装のぶれが別人化につながりやすいため、**EP001を人物固定の起点**として扱うことを明文化する。[1] [2]

この運用では、`assets/reference/` を正式保管庫、`assets/zopia_upload/season1_ep001_ep005/` をZopia手動アップロード用の出荷箱として分離したまま、必要な素材だけをコピーして使う。したがって、継続利用の中心にあるのは**ファイルの場所よりも、どの資産を「基準」と見なすか**という判断である。[2] [3]

## 2. 基本原則

実写版で固定するのは、**顔立ち、髪型、年齢感、服装系統、体格、役割関係**である。表情、視線、手の動き、姿勢、カメラ距離などは各話ごとに変えてよいが、それらはすべて**EP001基準資産から派生する差分**として扱う。[1]

| 項目 | 固定する内容 | 可変にしてよい内容 |
|---|---|---|
| 人物同一性 | 顔立ち、髪型、年齢感、体格 | 表情、目線、姿勢、演技の強さ |
| 衣装 | 基本服装系統 | しわ、立ち姿、細部の見え方 |
| 関係性 | 守屋守 = 受け手、相沢律子 = 補強、クマ課長 = 圧の発生源 | その回での登場比重、前景・後景の置き方 |
| 資産管理 | 既存キャラ差分として命名・追加 | 新規差分の作成自体 |

## 3. 共通参照資産の定義

Season 1前半で共通参照資産と見なすのは、**各キャラクターの基準顔・基準全身・主要姿勢差分**である。これらはEP001で人物認識を安定させるために整えたものであり、EP002〜EP005でも同じ人物を見せる起点として使う。[1] [3] [4]

| キャラクター | 主参照にする基準資産 | 補助参照として使う資産 |
|---|---|---|
| **守屋守** | `moriya_mamoru_base_face_front_v01.png`、`moriya_mamoru_base_front_v01.png` | `moriya_mamoru_pose_bust_45_v01.png`、`moriya_mamoru_pose_full_front_v01.png` |
| **相沢律子** | `aizawa_ritsuko_base_face_front_v01.png`、`aizawa_ritsuko_base_front_v01.png` | `aizawa_ritsuko_pose_bust_45_v01.png`、`aizawa_ritsuko_pose_full_front_v01.png` |
| **クマ課長** | `kuma_kacho_base_face_front_v01.png`、`kuma_kacho_base_front_v01.png` | `kuma_kacho_pose_bust_45_v01.png`、`kuma_kacho_pose_full_front_v01.png` |

背景とUIも前半5話では再利用資産として扱うが、人物同一性の主参照ではない。人物がぶれた場合は、まず背景や演出ではなく、**基準顔と基準全身の参照の仕方**を見直すべきである。[3]

## 4. 参照手順

実写版では、プロジェクト作成直後に基準素材を投入し、AI Agent に対して**「誰の基準画像か」「どの話でも同じ人物として使う資産か」**を明示する必要がある。最初に基準顔と基準全身を入れ、次に必要最小限の背景とUIを入れ、その後に表情差分・姿勢差分を足す順序を崩さない。[2] [3] [5]

| 手順 | 対象 | 意図 |
|---|---|---|
| 1 | 3名の基準顔 | 顔立ちを固定する |
| 2 | 3名の基準全身 | 服装・体格・全体シルエットを固定する |
| 3 | 3名の補助姿勢差分 | 構図の幅を増やす |
| 4 | 共通背景 | 場面の整合を取りやすくする |
| 5 | UI | 図鑑感や観測ログ演出を整える |
| 6 | 必要時のみ表情差分 | その話の症状表現を追加する |

## 5. 差分資産の扱い

新規表情差分や補助画像を追加する場合も、それは**既存キャラの差分**として命名し、主参照資産を置き換えるものではない。とくに前半5話で繰り返し使う差分名は、すでに `README_expression_filenames.md` に整理されている語彙へ合わせることで、エピソード横断の運用がぶれにくくなる。[4]

| キャラクター | 優先差分の考え方 | 代表例 |
|---|---|---|
| **守屋守** | 観測・停止・疲労の差分を優先する | `moriya_mamoru_expression_micro_stop_v01.png`、`moriya_mamoru_expression_fatigue_blank_v01.png` |
| **相沢律子** | 無言補強・やわらかい正当化・依頼微笑を優先する | `aizawa_ritsuko_expression_reinforce_silent_v01.png`、`aizawa_ritsuko_expression_reinforce_request_smile_v01.png` |
| **クマ課長** | ため息圧・静かな差し戻し・会議支配を優先する | `kuma_kacho_expression_pressure_sigh_v01.png`、`kuma_kacho_expression_soft_recheck_v01.png` |

## 6. 禁止事項と判断基準

実写版で避けるべきなのは、毎話ごとに新しい人物像を発明してしまう運用である。もし新しい画像が必要になっても、まず**「これはEP001基準人物の差分か、それとも別人か」**を判定しなければならない。差分として説明できるなら追加候補になり、説明できないなら採用を見送る。[1] [4]

| 判断場面 | 採用してよい例 | 避けるべき例 |
|---|---|---|
| 顔の差分 | 同一人物として自然な表情変化 | 年齢感や骨格が大きく変わる別人化 |
| 服装差分 | 同系統のオフィス服の範囲内 | キャラ印象を変える大幅衣装変更 |
| ポーズ差分 | 半身、全身、軽い角度違い | キャラ性を壊す過剰な演技姿勢 |
| 追加生成 | 既存命名規則に沿った差分追加 | 新キャラ扱いで別フォルダ運用 |
