# ショート動画ロゴ表示ルール

## 目的

本メモは、**職場ウイルス図鑑**のショート動画におけるロゴ表示の基本ルールを定めるための運用整理である。シリーズ認知を維持しつつ、30秒前後の短尺映像で最も重要な**人物の表情、字幕、図鑑情報**を邪魔しないことを優先する。

## 基本方針

ショート動画におけるロゴは、主役として大きく見せるのではなく、**シリーズ識別用の常設サイン**として扱う。したがって、本編中は右上に控えめに常時表示することを標準とし、視聴体験の中心はあくまでエピソード内容に置く。

| 項目 | 標準ルール |
|---|---|
| 表示位置 | 画面右上固定 |
| 表示方式 | 本編中は常時表示 |
| サイズ目安 | 画面幅の約10〜15% |
| 不透明度目安 | 80〜90% |
| 優先順位 | 字幕・顔・図鑑カードより下げる |

## 実務上の判断基準

右上固定を基本とする理由は、縦型ショート動画で中央帯と下部帯に情報が集中しやすく、右上が比較的干渉を起こしにくいためである。ただし、人物の頭部、強いリアクション、強調字幕、図鑑カードが右上に来るカットでは、一時的に少し縮小するか、数フレーム単位で位置を微調整してよい。重要なのは、ロゴを守ることではなく、**本編の可読性と感情伝達を守ったうえでロゴを自然に残すこと**である。

| NGに近い配置 | 理由 |
|---|---|
| 顔の真横に大きく重なる配置 | 表情差分の読解を妨げる |
| 強調字幕の直上・直下に密着する配置 | 文字の視認性が落ちる |
| 図鑑カードや診断テロップと競合する配置 | 情報優先順位が崩れる |
| 高彩度・高不透明度で目立ちすぎる表示 | 本編よりロゴが先に目に入る |

## 冒頭・締めでの扱い

本編中は控えめ表示を基本とする一方、必要に応じて**冒頭または締めのいずれか一回のみ**、通常よりやや大きめにロゴを見せてもよい。ただし、毎回必須ではない。タイトル認知を補強したい回、シリーズ導線を強めたい回に限定して使う。

| 場面 | 推奨運用 |
|---|---|
| 本編中 | 右上に小さく常設 |
| 冒頭 | 必要時のみ少し大きめに1回表示 |
| 締め | 必要時のみエンドカード的に表示 |
| 毎回の派手なアニメーション | 原則不要 |

## 使用ロゴとファイル

ショート動画の標準ロゴは、以下のメインファイルを使用する。

| 用途 | 推奨ファイル |
|---|---|
| 標準表示用ロゴ | `assets/reference/brand/shokuba_virus_zukan_logo_main_v01.png` |

背景との干渉が強い場合に備えて、将来的に透過版や簡易版を追加してもよいが、現段階ではメインロゴを縮小表示する運用で十分とする。

## 運用メモ

実編集では、ロゴは**常設ウォーターマークに近い扱い**と考えると判断しやすい。すなわち、常に存在は認識できるが、視聴者の注意を奪わない状態が理想である。職場ウイルス図鑑は、派手なエンタメ性と静かな職場怪談感の中間にあるため、ロゴの扱いも同様に**控えめだが記憶には残る**状態を目指す。
