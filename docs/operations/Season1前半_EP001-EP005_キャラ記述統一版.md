# Season 1前半 EP001〜EP005 キャラ記述統一版

## 1. この文書の位置付け

本書は、**EP001〜EP005の各エピソードシートで使うキャラクター記述の統一基準**を明文化したものである。目的は、各話ごとの症状や演出を変えながらも、**守屋守・相沢律子・クマ課長の役割定義がぶれない状態**を保つことにある。[1] [2]

すでに前半5話では、主人公軸、圧力源、静かな補強役の配置を寄せる調整が進んでいる。本書では、その整理結果を「各EPでどう書くべきか」という運用レベルまで落とし込み、EPシート更新時の判断基準として再利用できるようにする。[2]

## 2. 共通記述の中核

各EPでは、登場比重や症状の見せ方は変えてよいが、以下の三点は維持する。第一に、**守屋守は常に観測者であり、受け手の中心人物であること**。第二に、**相沢律子は露骨な敵役ではなく、静かに場を補強する人物であること**。第三に、**クマ課長は管理職側の圧の起点であり、直接命令しない場面でも構造圧を発生させること**である。[1] [2]

| キャラクター | 統一して書くべき基準文意 | 各EPで変えてよい文意 |
|---|---|---|
| **守屋守** | 現場寄りの中堅社員であり、違和感を最も強く受ける観測者 | どの症状をどう受けるか、疲労や戸惑いの強度 |
| **相沢律子** | 静かな補強者であり、理性的・感じの良い言動で異常を正常化する | 無言同調、正当化、依頼、流し方の種類 |
| **クマ課長** | 圧の発生源であり、管理職側の象徴として場を止める | ため息、差し戻し、会議支配、口頭圧、遠景圧など圧の型 |

## 3. EPごとの統一方針

EPごとの差は、あくまでウイルスの症状と場面の差であり、キャラの根本定義の差ではない。各回では、以下のように**同じ三角構造の中で症状だけを変える**書き方へ揃える。[1] [2]

| 話数 | 守屋守 | 相沢律子 | クマ課長 | 統一上のポイント |
|---|---|---|---|---|
| **EP001 ため息ウイルス** | 空気停止を最初に観測する | 無言で緊張を補強する | ため息で場を止める | 導入回として3名の関係を最も明瞭に見せる |
| **EP002 差し戻し確認ウイルス** | 終わらない確認の受け手になる | 「もう一度確認」を自然化する | 理由なき差し戻しを繰り返す | 確認圧の構造を三角関係で見せる |
| **EP003 責任転嫁ウイルス** | 会議で引き受け役にされる | 理性的な言葉で責任移動を補強する | 会議の方向を決めて責任を流す | 責任の流れだけが自然化する怖さを保つ |
| **EP004 口頭指示ウイルス** | 記録外の指示を受け手として抱える | 介入せず通常化を補強する | 軽い口頭指示を既成事実化する | 記録不在と構造圧の結びつきを明確にする |
| **EP005 なぜか君担当ウイルス** | 小さな依頼の受け皿になる | 丁寧な依頼で偏りを成立させる | 断りづらい空気を残す | 悪人不在でも偏りが固定される構造を保つ |

## 4. エピソードシート更新時の記述ルール

EPシートの更新では、キャラクターの説明を各話ごとに完全新規で書き直すのではなく、**共通定義を土台に、その回の症状に応じた補足だけを足す**方針を取るのがよい。これにより、シリーズ性と制作効率の両方を維持できる。[1] [3]

| 記述箇所 | 共通で維持すべき内容 | 各話で追加する内容 |
|---|---|---|
| 主人公欄 | 守屋守 = 観測者 / 受け手の中心人物 | 今回どの症状を受けるか |
| 相手側欄 | クマ課長 = 圧の発生源 | 今回の圧の型 |
| 周囲欄 | 相沢律子 = 静かな補強者 | 今回の補強の仕方 |
| 展開メモ | 三角構造は維持 | 場面、台詞、カット上の違い |

## 5. 既存EPシートへの反映方針

今回の反映では、EP001〜EP005の各ファイルに**「Season 1前半レギュラー固定の前提」**を明示し、シリーズ前半で固定している役割関係を読み手が見失わないようにする。この追記は、各EP固有の物語を壊さずに、共通運用の土台だけを補強する最小修正として位置付ける。[1]
