# 職場ウイルス図鑑｜Season 1〜Season 4 演出・カット・セリフ標準ルール

## 1. この文書の役割

本書は、Season 1〜Season 4 の各話制作で共通に参照する**演出ルール、カット構成、セリフテンプレ、情報開示の標準**を定めるための文書である。今回の改訂では、シリーズの核を「段階公開方式」に置き直し、映像で見せる情報量をSeasonごとに制御する方針へ更新した。[1] [2]

シリーズはシーズンごとに緊張度と可視化範囲が変化するが、ショート動画としての見やすさ、連鎖視聴性、現実感の維持は全Seasonで共通して守る必要がある。したがって、各話制作ではまず**何を見せるSeasonか**を確認し、そのうえで演出強度を決める。

## 2. 共通演出ルール

シリーズ全体では、何よりも**過剰説明より空気で見せること**を優先する。ショート動画である以上、設定説明の長台詞よりも、視線、沈黙、周囲の平然さ、ログ表示、机上の残務、暗転の切れ味などで見せた方が強い。各Seasonで情報量は増えるが、映像の基本姿勢は変えない。[2]

| 項目 | 共通標準 | 禁止寄り判断 |
|---|---|---|
| リアリティ | 現代日本の会社に見えることを優先する | 完全ファンタジー化、過度な超常表現 |
| 説明量 | 最小限に抑え、視覚情報で補う | 設定を口頭で長く説明する |
| 空気表現 | 沈黙、間、周囲の平然さを使う | 何でも台詞で言わせる |
| 情報開示 | Seasonごとに必要な情報だけ見せる | 早いSeasonで全部見せる |
| 結末 | 次が気になる形で閉じる | 毎回完全解決する |

## 3. 情報開示の演出原則

段階公開方式では、世界観の設定量ではなく、**視聴者に見せる表層情報**を管理する。演出はこの順序に従うことで、Season 1 の入口の強さを維持しつつ、後続Seasonで理解の深度を増していく。[1]

| 要素 | Season1 | Season2 | Season3 | Season4 |
|---|---|---|---|---|
| ウイルス名 | 表示 | 表示 | 表示 | 表示 |
| 型 | 表示 | 表示 | 表示 | 表示 |
| 株 | 非表示 | 表示 | 表示 | 表示 |
| ワクチン | 最小限 | 最小限 | 表示 | 表示 |
| 感染レベル | 非表示 | 非表示 | 裏設定 | 表示 |
| 上司分布 | 非表示 | 非表示 | 非表示 | 表示 |

## 4. Season別演出ルール

### 4-1. Season 1

Season 1 は、**観測と命名のSeason**である。そのため、演出は最も抑制的でなければならない。リアル寄りの職場空気、微妙な顔の止まり、誰も異常扱いしない周囲の反応が主役であり、エフェクトは最小限に留める。[1]

| 項目 | Season 1 標準 |
|---|---|
| 画のトーン | 現実のオフィス、自然光または通常の室内光 |
| 音の使い方 | 生活音、タイピング音、短い通知音、沈黙 |
| UI量 | ほぼなし。命名時のみ最小限 |
| 表示情報 | ウイルス名、型のみ |
| 株の扱い | 非表示 |
| ワクチンの扱い | 原則非表示、必要時のみ補助的 |
| 締め方 | 不穏さを残す、解決しない |

### 4-2. Season 2

Season 2 では、同じウイルスに複数の株があることが見え始める。分類が深くなるが、まだ防衛策を主役にしない。視聴者が違いに気づき、コメント欄で命名遊びをしたくなる見せ方が重要である。[1] [2]

| 項目 | Season 2 標準 |
|---|---|
| 画のトーン | 現実寄りのまま、比較・分類の視点を強める |
| 音の使い方 | 生活音中心、軽い分類感を持たせる反復演出可 |
| UI量 | 最小限の分類表示を追加可能 |
| 表示情報 | ウイルス名、型、株 |
| ワクチンの扱い | 最小限 |
| 締め方 | これは同じ型でも一種類ではない、と示す |

### 4-3. Season 3

Season 3 では、防衛策を前面化する。ここで初めてワクチンが**使える言葉**として立ち上がり、リーマン自己防衛システムとの接続が見える。ただし、万能の解決手段としては扱わず、現実寄りの小さな防衛として描く。[1] [2]

| 項目 | Season 3 標準 |
|---|---|
| 画のトーン | 現実寄りのまま整理度と実用性を上げる |
| 音の使い方 | 低い作動音、通知音、静かな防衛感 |
| UI量 | モジュール名、簡易ログ、補助表示を追加 |
| 表示情報 | ウイルス名、型、株、ワクチン |
| 感染レベル | 裏設定運用。原則表示しない |
| 締め方 | 防げるが、まだ完全ではない |

### 4-4. Season 4

Season 4 では、感染レベルと上司別分布まで見せ、図鑑世界を完成させる。ここでは個別ネタを超えて、構造、流行、分布、全体傾向が視聴者に見える必要がある。[1] [2]

| 項目 | Season 4 標準 |
|---|---|
| 画のトーン | 図鑑・分析・資料としても耐える整理度 |
| 音の使い方 | 通知、全体俯瞰、構造可視化に合う静かな緊張 |
| UI量 | 感染レベル、分布図、分類情報を表示可能 |
| 表示情報 | ウイルス名、型、株、ワクチン、感染レベル、上司分布 |
| 締め方 | 個人の違和感が職場全体の流行構造へ接続する |

## 5. カット構成の標準

ショート動画としての回転性を保つため、各Seasonのカット構成は**骨格を共通化しつつ、見せる情報だけ変える**方がよい。違いは、何を中心に据えるかであり、導入・異変・認識・締めの骨格は維持する。[2]

| Season | 推奨カット数 | 標準構成 |
|---|---|---|
| **Season 1** | 5カット前後 | 違和感 → 感染 → セリフ発症 → 命名 → 未解決 |
| **Season 2** | 5〜6カット | 違和感 → 感染 → セリフ発症 → 命名 + 株示唆 → 未解決 |
| **Season 3** | 6〜7カット | 違和感 → 感染 → セリフ発症 → 命名 + 株 → ワクチン導入 → 不完全防衛 |
| **Season 4** | 6〜8カット | 違和感 → 感染 → セリフ発症 → 命名 + 株 + ワクチン → 感染レベル / 分布可視化 → 構造提示 |

## 6. Season 1 固定テンプレート

Season 1 では、全EPで共通の流れを固定する。ここで揺らぐと、シリーズ入口としてのわかりやすさが崩れるため、例外回をつくる場合もこのテンプレートから逸脱しない方がよい。[1]

| 段階 | 役割 | 内容の目安 |
|---|---|---|
| 1 | 違和感（レベル1） | 小さな異常を見せる。まだ説明しない |
| 2 | 感染（レベル2） | 守屋守が巻き込まれ始める |
| 3 | セリフ発症 | そのウイルスを象徴する日常語が出る |
| 4 | ウイルス命名 | **ウイルス名 + 型**のみ表示する |
| 5 | 未解決で終了 | 解決せず、不穏さだけを残す |

このテンプレートでは、感染レベルの概念を映像内に表示しない。レベルはあくまで内部演出管理のための裏設定として使用し、視聴者体験は「気づき」で止める。

| 6 | 上司分布 / 構造可視化 | 個人事象を構造へつなげる |
| 7 | 締め | これは流行構造であると示す |

## 8. セリフテンプレートの標準

台詞は、世界観説明ではなく**圧の自然化、違和感の立ち上がり、分類の精度、防衛の限界**を見せるために使う。説明過多を避けるため、台詞テンプレートは短く、用途別に使い分ける。[2]

### 8-1. Season 1 セリフテンプレ

| 用途 | テンプレ例 |
|---|---|
| 異変の一撃 | 「もう一回見ておいて」 / 「さっき言ったよね」 / 「……はぁ」 |
| 周囲の正常化 | 「まあ、あるよね」 / 「確認だけお願いします」 |
| 観測者の内面 | 「……あれ？」 / 「今の、何だ？」 / 「気のせいか？」 |
| 締め | 「これは、ただの違和感ではなかった」 / 「何かがおかしい…」 / 「気のせい…なのか？」 |

### 8-2. Season 2 セリフテンプレ

| 用途 | テンプレ例 |
|---|---|
| 分類気づき | 「同じじゃない」 / 「また別の出方だ」 |
| 株示唆 | 「この前と違う」 / 「こっちはその言い方か」 |
| 締め | 「同じウイルスでも、出方が違う」 / 「これは別株かもしれない」 |

### 8-3. Season 3 セリフテンプレ

| 用途 | テンプレ例 |
|---|---|
| 防衛導入 | 「先に条件を残しておこう」 / 「今のうちに記録しておく」 |
| モジュール名 | 「議事録バリアを入れる」 / 「証跡ログを残す」 |
| 締め | 「防げる…だが、完全ではない」 / 「次はもう少し早く守れる」 |

### 8-4. Season 4 セリフテンプレ

| 用途 | テンプレ例 |
|---|---|
| 構造化 | 「個人の問題じゃない」 / 「部署をまたいで広がっている」 |
| 可視化 | 「感染レベルが上がっている」 / 「この上司群に偏っている」 |
| 締め | 「これは職場全体の流行だ」 / 「もう、見えないふりはできない」 |

## 9. 運用ルール

今後の全話制作では、まずSeasonを決め、その次に本書のSeason別ルールに沿って**表示情報、演出強度、カット数、台詞量、締めの文体**を選ぶ。Season 1 の回で株や感染レベルを前面に出したり、Season 3 の回で唐突に完全勝利させたりすることは避ける。[1] [2]

また、例外回を作る場合でも、例外にする理由を文書内で明記する。シリーズ性を壊さない限りでの変奏は許容されるが、**標準を崩す場合は意識的に崩す**ことを原則にする。
