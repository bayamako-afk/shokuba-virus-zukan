# 職場ウイルス図鑑｜Season 1〜Season 3 演出・カット・セリフ標準ルール

## 1. この文書の役割

本書は、Season 1〜Season 3 の各話制作で共通に参照する**演出ルール、カット構成、セリフテンプレの標準**を定めるための文書である。シリーズはシーズンごとに緊張度が変化するが、動画としての見やすさ、連鎖視聴性、現実感の維持は全シーズンで共通して守る必要がある。

したがって、各話の制作ではまず本書で**共通ルール**を確認し、そのうえでSeason別のテンプレートへ具体化する。演出は強めてもよいが、常に「現実の会社で本当にありそうな延長」に見えることを基準に判断する。

## 2. 共通演出ルール

シリーズ全体では、何よりも**過剰説明より空気で見せること**を優先する。ショート動画である以上、設定説明の長台詞よりも、視線、沈黙、周囲の平然さ、ログ表示、机上の残務、暗転の切れ味などで見せた方が強い。各Seasonで情報量は増えるが、映像の基本姿勢は変えない。

| 項目 | 共通標準 | 禁止寄り判断 |
|---|---|---|
| リアリティ | 現代日本の会社に見えることを優先する | 完全ファンタジー化、過度な超常表現 |
| 説明量 | 最小限に抑え、視覚情報で補う | 設定を口頭で長く説明する |
| 空気表現 | 沈黙、間、周囲の平然さを使う | 何でも台詞で言わせる |
| エフェクト | Seasonに応じて段階的に増やす | 初期から派手に盛りすぎる |
| 結末 | 次が気になる形で閉じる | 毎回完全解決する |

## 3. Season別演出ルール

### 3-1. Season 1

Season 1 は、観測と命名のシーズンである。そのため、演出は最も抑制的でなければならない。リアル寄りの職場空気、微妙な顔の止まり、誰も異常扱いしない周囲の反応が主役であり、エフェクトは最小限に留める。

| 項目 | Season 1 標準 |
|---|---|
| 画のトーン | 現実のオフィス、自然光または通常の室内光 |
| 音の使い方 | 生活音、タイピング音、短い通知音、沈黙 |
| UI量 | ほぼなし、または命名時のみ最小限 |
| 感情表現 | 飲み込み、停止、目線、微妙な表情変化 |
| 締め方 | 不穏さを残す、解決しない |

### 3-2. Season 2

Season 2 では、現実の職場に加えて自宅AI司令室が加わるため、視覚情報が一段増える。ただし、デジタルUIやログ表示は**分析の補助**であって、派手なSF演出のために使うのではない。職場のリアリティと、自宅での静かな記録・分析が自然に接続することが重要である。

| 項目 | Season 2 標準 |
|---|---|
| 画のトーン | 職場は現実寄り、自宅は静かな分析空間 |
| 音の使い方 | キーボード、通知、低い作動音、控えめなUI音 |
| UI量 | ログ、シールド名、簡易モジュール表示を追加 |
| 感情表現 | 観測に加え、分析、警戒、静かな決意 |
| 締め方 | 防げたが完全ではない、次に備える |

### 3-3. Season 3

Season 3 は、複数感染や組織構造を扱うため、演出スケールを拡大してよい。ただし、その拡大は映像的派手さではなく、**構造の見え方**として行うのが基本である。感染源マップ、理不尽チェーン解析、同時発生の見せ方などは、視聴者が「会社全体の問題だ」と理解できる粒度で整理する。

| 項目 | Season 3 標準 |
|---|---|
| 画のトーン | 現実ベースのまま緊張感を強める |
| 音の使い方 | 通知、警戒感のある低音、切迫した静けさ |
| UI量 | マップ、クラスター、複合検知などを追加 |
| 感情表現 | 覚悟、判断、共有、備え |
| 締め方 | 戦いの継続、後戻りできなさ |

## 4. カット構成の標準

ショート動画としての回転性を保つため、各Seasonのカット構成は**大きく変えすぎない**方がよい。違いは、何を中心に据えるかであり、導入・異変・認識・締めの骨格は維持する。

| Season | 推奨カット数 | 標準構成 |
|---|---|---|
| **Season 1** | 5〜6カット | 導入 → 異変 → 周囲の平然 → 観測者の引っかかり → 命名 |
| **Season 2** | 6〜7カット | 導入 → 異変 → 観測 → 自宅分析 / AI補助 → 小規模防衛 → 不完全な締め |
| **Season 3** | 6〜8カット | 導入 → 複合異変 → 他者巻き込み → 構造可視化 → 備え / 対処 → 戦闘継続の締め |

さらに、尺別に見たときの基準は次のとおりである。

| 尺 | 基本用途 | 使い方 |
|---|---|---|
| **15秒** | 先行出し、告知、現象の切り出し | 1異変を一撃で見せる |
| **30秒** | 主力尺 | そのSeasonの型を最も素直に見せる |
| **45秒** | 当たり回、説明が少し必要な回 | 世界観や構造を少し厚く見せる |

## 5. Season別カットテンプレート

### 5-1. Season 1

Season 1 では、**職場だけで閉じる**構成を基本とする。観測者が違和感を受け取り、周囲が正常化し、命名または不穏な余韻で終える。

| Cut | 役割 | 内容の目安 |
|---|---|---|
| 1 | 導入 | 普通の職場、普通の会話、普通の手順 |
| 2 | 異変 | ため息、差し戻し、口頭圧などの一撃 |
| 3 | 周囲 | 誰も異常扱いしない |
| 4 | 観測 | 守屋守の停止、視線、飲み込み |
| 5 | 締め | 命名、暗転、違和感の残響 |

### 5-2. Season 2

Season 2 では、職場で起きた現象が**自宅分析パートへ接続**することが重要である。防衛モジュールは見せるが、万能感を出してはいけない。

| Cut | 役割 | 内容の目安 |
|---|---|---|
| 1 | 導入 | 職場の日常とウイルス発生 |
| 2 | 異変 | 主人公が被弾または観測 |
| 3 | 記録 | ログ化、メモ、違和感の整理 |
| 4 | 分析 | 自宅AI司令室で構造把握 |
| 5 | 防衛 | 1モジュールだけ試す |
| 6 | 締め | 効果はあるが不完全 |

### 5-3. Season 3

Season 3 では、主人公だけでなく**他者と構造**も画面に入れる。被感染者、観測者、圧の発生源、組織マップがつながって見える構成が望ましい。

| Cut | 役割 | 内容の目安 |
|---|---|---|
| 1 | 導入 | 一見普通だが不穏な組織空気 |
| 2 | 複合異変 | 複数のウイルス兆候が重なる |
| 3 | 他者巻き込み | 別の被感染者や観測者が出る |
| 4 | 構造可視化 | マップ、クラスター、チェーン解析 |
| 5 | 備え / 対処 | 主人公が守る側として動く |
| 6 | 締め | 戦いは続く、後戻りできない |

## 6. セリフテンプレートの標準

台詞は、世界観説明ではなく**圧の自然化、違和感の立ち上がり、対処の不足感**を見せるために使う。説明過多を避けるため、台詞テンプレートは短く、用途別に使い分ける。

### 6-1. Season 1 セリフテンプレ

Season 1 では、台詞はほぼ日常会話に留める。視聴者が怖がるのは異常な言葉そのものではなく、**普通の言い方なのにおかしいこと**だからである。

| 用途 | テンプレ例 |
|---|---|
| 異変の一撃 | 「もう一回見ておいて」 / 「さっき言ったよね」 / 「……はぁ」 |
| 周囲の正常化 | 「まあ、あるよね」 / 「確認だけお願いします」 |
| 観測者の内面 | 「……あれ？」 / 「今の、何だ？」 / 「気のせいか？」 |
| 締め | 「これは、ただの違和感ではなかった」 / 「何かがおかしい…」 / 「気のせい…なのか？」 |

### 6-2. Season 2 セリフテンプレ

Season 2 では、観測語彙に加えて分析語彙と防衛語彙が入る。ただし、AIやモジュールの名称は短く扱い、専門用語のように長く説明しない。

| 用途 | テンプレ例 |
|---|---|
| 観測ログ | 「記録する」 / 「また同じ型だ」 / 「条件が見えた」 |
| AI相棒 | 「ログを照合します」 / 「発症条件を推定します」 |
| 防衛モジュール | 「議事録バリアを展開」 / 「証跡ログを保存」 |
| 締め | 「防げる…だが、完全ではない」 / 「これは一部に過ぎない」 |

### 6-3. Season 3 セリフテンプレ

Season 3 では、視点が主人公個人から組織へ広がるため、共有、警戒、備えの語彙が増える。ここでも大げさな戦闘口調にはせず、現実の言葉に近い範囲で緊張を出す。

| 用途 | テンプレ例 |
|---|---|
| 複合検知 | 「一つじゃない」 / 「別部署でも出ている」 |
| 共有 / 指示 | 「この型を共有して」 / 「そのログを残しておいて」 |
| 備え | 「次は守る側に回る」 / 「ここで止める必要がある」 |
| 締め | 「これは戦いだ」 / 「もう戻れない」 |

## 7. 運用ルール

今後の全話制作では、まずSeasonを決め、その次に本書のSeason別ルールに沿って**演出強度、カット数、台詞量、締めの文体**を選ぶ。Season 1 の回に Season 3 の警戒演出を持ち込んだり、Season 2 の回で初期から完全勝利させたりすることは避ける。

また、例外回を作る場合でも、例外にする理由を文書内で明記する。シリーズ性を壊さない限りでの変奏は許容されるが、**標準を崩す場合は意識的に崩す**ことを原則にする。
