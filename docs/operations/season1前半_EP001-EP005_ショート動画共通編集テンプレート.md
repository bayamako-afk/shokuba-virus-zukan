# Season 1前半 EP001〜EP005 ショート動画共通編集テンプレート

## 目的

本テンプレートは、**職場ウイルス図鑑** Season 1前半のEP001〜EP005を、同じシリーズ作品として連続運用するための**共通編集基準**をまとめたものである。すでに整備済みの30秒shotlist、必要素材一覧、実写コンテ用コマ割り指示、キャラ固定方針、ロゴ表示ルールを、実際の編集タイムラインへ接続できる単位に整理し直し、各話の差分作業を最小化することを狙う。

本テンプレートの役割は、各話の物語内容を均一化することではない。むしろ、**画面構造・字幕運用・ロゴ運用・命名演出・音の置き方を共通化し、エピソードごとの違いはショット内容と表情差分で見せる**ための基盤である。

## 想定フォーマット

EP001〜EP005のショート動画は、縦型の短尺視聴を前提とし、情報密度が高くても視線が迷いにくい構造にする。

| 項目 | 標準値 | 備考 |
|---|---|---|
| 画角 | 9:16 縦型 | ショート動画前提 |
| 解像度 | 1080 × 1920 | 編集・書き出し共通基準 |
| 尺 | 約30秒 | 28〜32秒程度の微差は許容 |
| 基本ショット数 | 6〜7ショット | 既存shotlistに準拠 |
| 編集トーン | 実務的な静けさ + じわ怖さ | 過剰に派手にしない |
| 主視点 | 守屋守 | 前半5話共通 |

## 編集の基本思想

画面の中心は、常に**「誰が空気を受けているか」**が分かる状態に保つ。したがって、ショート動画であってもテンポ重視の高速編集に寄せすぎず、**止まる視線、飲み込む間、軽い沈黙、空気の固定**を残すことが重要である。クマ課長の圧や相沢律子の補強は、大きなアクションではなく、守屋守の反応によって怖さが立ち上がるため、編集は情報整理よりも**反応の読ませ方**を優先する。

また、シリーズとしては同じ職場で起きている感覚を維持する必要があるため、背景、字幕位置、命名演出、ロゴ位置、SEの基調を毎話で大きく変えない。差分は、エピソード固有のUI、表情差分、ショット内の圧の種類で出す。

## 共通タイムライン構成

30秒前後の尺は、以下の5ブロックで考えると編集判断が安定する。

| ブロック | 秒数目安 | 役割 | 編集上の考え方 |
|---|---:|---|---|
| A. 導入 | 0〜4秒 | 職場の日常提示 | 同じ会社感を一目で入れる |
| B. 異常発火 | 4〜10秒 | 小さな違和感の発生 | クマ課長または相沢律子の作用を見せる |
| C. 伝播 | 10〜20秒 | 空気が構造化される | 守屋守の反応を中心に置く |
| D. 固定化 | 20〜26秒 | 異常が通常化する | 周囲の止めなさを入れる |
| E. 命名締め | 26〜30秒 | 図鑑化・観測ログ化 | シリーズ記号として固定 |

この5ブロックは、全話で厳密に秒単位固定する必要はないが、**導入 → 違和感 → 構造化 → 固定化 → 命名**の流れは維持する。とくに最後の命名締めは、シリーズ識別上の重要な固定パーツであるため、演出を大きく揺らしすぎない。

## 画面レイヤー構成

実編集時は、素材を感覚的に重ねるのではなく、以下のレイヤー順で管理すると差し替えが容易になる。

| レイヤー | 内容 | 運用 |
|---|---|---|
| L1 背景 | 執務室、会議室、廊下など | 共通背景を話ごとに差し替え |
| L2 人物ベース | 守屋守、クマ課長、相沢律子 | 基準立ち絵・半身素材 |
| L3 人物差分 | 表情、視線、姿勢差分 | 話ごとに差し替える主要層 |
| L4 小物 / UI | チャット、書類、メモ、付箋など | 物語説明に必要な情報層 |
| L5 字幕 | セリフ、観測補助テキスト | 可読性最優先 |
| L6 ロゴ | 右上常設 | 本編より目立たせない |
| L7 命名演出 | ai_log / registry card | 終盤のみ強く出す |

## カット接続ルール

各ショットは、派手なトランジションよりも、**空気の継続が切れない接続**を優先する。基本はカット切り替え、必要に応じて短いディゾルブ、暗転、UI被せを使う。ズームや回転、強いグリッチは多用しない。

| 接続方法 | 使用場面 | ルール |
|---|---|---|
| 直カット | 通常ショット接続 | 基本形 |
| 短ディゾルブ | 空気がじわっと伝播する場面 | 0.1〜0.3秒程度まで |
| テキスト被せカット | 通知・命名・理解の瞬間 | UI優先回のみ |
| 暗転 | 命名締め前後 | 毎話の固定パーツに近い扱い |
| 派手なモーション | 原則不使用 | 本編のじわ怖さを損なうため |

## 字幕テンプレート

字幕は、情報説明ではなく感情の補助として置く。守屋守の違和感を先回りして全部説明しないことが重要である。テキストが多すぎると、短尺の不気味さより説明動画に寄ってしまう。

| 項目 | 標準ルール |
|---|---|
| 字幕位置 | 下部中央寄せ |
| 行数 | 1〜2行まで |
| 文字量 | 1カット内で短文中心 |
| 役割 | セリフ補助 + 最小限の意味固定 |
| 強調表現 | 必要語のみ色替え or 太字相当 |
| 禁止傾向 | 長文解説、感情の説明しすぎ |

字幕の優先順位は、**セリフ > 状況理解 > 命名**の順である。沈黙が効く場面では、あえて字幕を減らし、表情差分と間で読ませる。

## ロゴ運用

ロゴは既定の運用どおり、**右上固定の控えめ常設**を標準とする。ロゴはブランド認知のために存在するが、人物の顔、字幕、図鑑カードを邪魔してはならない。

| 項目 | 標準ルール |
|---|---|
| 使用ロゴ | `assets/reference/brand/shokuba_virus_zukan_logo_main_v01.png` |
| 基本位置 | 右上固定 |
| サイズ感 | 画面幅の10〜15% |
| 不透明度 | 80〜90% |
| 例外対応 | 顔・字幕・カードと重なる場合のみ微調整 |

冒頭または締めで少し大きめに見せる運用は可能だが、毎話の必須処理にはしない。まずは常設ロゴだけで成立する編集テンプレートとして組む。

## 命名演出テンプレート

各話の終わりは、**観測の結果として命名される**感覚が出ることが重要である。したがって、終盤は演出を毎話バラバラにせず、以下のどちらかの型に寄せる。

| 命名型 | 使用素材 | 向いている話 |
|---|---|---|
| 観測ログ型 | `ai_log_overlay_v1` | EP001, EP002, EP004, EP005 |
| 図鑑カード型 | `registry_card_v1` | EP003 |

命名直前では、守屋守の**micro_理解**や**fatigue**系差分を最後に置き、そこから暗転または画面固定で命名へ移る。ここで急に勢いをつけすぎず、**気づいてしまった感じ**で終えるのが基調である。

## 音編集テンプレート

音は、派手なBGM主導ではなく、**環境音と小さなSEで気味悪さを積む**方向を標準とする。職場ウイルス図鑑の前半5話は、音で怖がらせるより、普通の職場音が少しだけ冷える感覚を重視する。

| 音要素 | 標準運用 | 備考 |
|---|---|---|
| 環境音 | 薄く常設 | タイピング、空調、紙、椅子など |
| ため息・通知・紙音 | ピンポイントで強調 | EPごとの差分の起点になる |
| BGM | 低主張 | 空気を壊さないこと優先 |
| 無音 | 要所で使用 | 命名前や違和感瞬間で有効 |
| 過剰SE | 原則避ける | ホラー記号を強くしすぎない |

## 再利用コンポジション案

編集を速くするため、素材単位ではなく**コンポジション単位**で共通部品を作ることを推奨する。

| コンポ名 | 内容 | 用途 |
|---|---|---|
| `comp_intro_office_v1` | 導入用の職場全景 | EP001〜EP005の冒頭土台 |
| `comp_reaction_moriya_v1` | 守屋守リアクション寄り | 各話の違和感反応に再利用 |
| `comp_pressure_kuma_v1` | クマ課長圧ショット | ため息、差し戻し、軽口指示に流用 |
| `comp_reinforce_aizawa_v1` | 相沢律子補強ショット | 無言同調、補強セリフ、依頼微笑に流用 |
| `comp_naming_ai_log_v1` | 観測ログ締め | EP001, EP002, EP004, EP005 |
| `comp_naming_registry_v1` | 図鑑カード締め | EP003 |
| `comp_logo_watermark_v1` | 右上ロゴ常設 | 全話共通 |

## 話ごとの差分適用ポイント

共通テンプレートを使う場合でも、各話で差し替える箇所は明確にしておく必要がある。

| 話数 | 主差分 | 変更すべき主要要素 |
|---|---|---|
| EP001 | ため息で空気が止まる | クマ課長差分、朝執務室、ため息SE |
| EP002 | 理由なき差し戻し | チャットUI、守屋守通知停止、相沢補強セリフ |
| EP003 | 会議で責任が流れる | 会議室背景、図鑑カード締め、相沢理性的補強 |
| EP004 | 記録なき口頭指示 | 廊下背景、歩行ショット、メモ不在演出 |
| EP005 | 感じよく偏りが積まれる | 夕方執務室、付箋・残務、相沢依頼微笑 |

## 編集開始前チェックリスト

編集着手前は、各話の内容確認よりも、共通部品が揃っているかの確認を先に行う方が効率的である。

| 確認項目 | チェック内容 |
|---|---|
| ロゴ | 右上常設用ロゴが読み込まれているか |
| 人物素材 | 3人のbaseと必要差分が揃っているか |
| 背景 | 話数に応じた共通背景が揃っているか |
| UI | ai_log、chat、registry card など必要UIがあるか |
| 字幕 | 位置・サイズ・色のプリセットができているか |
| 音 | 環境音の基調と必要SEが整理されているか |

## 運用結論

EP001〜EP005の共通編集テンプレートは、**同じ職場で同じ3人を観測し続けるシリーズ感**を維持するための骨組みである。したがって、毎話で新しい見せ方を探すよりも、**守屋守の反応を読ませる中央設計、右上ロゴ常設、終盤の命名固定、静かな職場音、過剰でない字幕**を安定運用することが優先される。

このテンプレートに沿って編集すれば、各話は差分だけを差し替えて成立しやすくなり、前半5話を通した世界観の連続性も維持しやすい。
