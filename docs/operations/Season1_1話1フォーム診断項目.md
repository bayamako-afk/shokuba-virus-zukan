# 職場ウイルス図鑑 Season1 1話1フォーム診断項目

## 1. 文書の目的

本書は、『職場ウイルス図鑑 Season1』の EP001〜EP010 について、各話ごとに**1話1フォーム・3問構成**の簡易チェックを実装できるようにするための運用文書である。目的は、視聴者が動画視聴後に気軽に参加できる**職場あるあるチェック**を用意し、コメント、概要欄、外部フォーム、将来の「職場ウイルス度チェック」本体へ接続しやすい診断素材として整備することにある。[1] [2]

本書で扱うフォームは、医療的、心理学的、メンタルヘルス的な診断ではない。あくまで創作コンテンツ連動の軽い参加施策として設計し、結果も断定ではなく**「潜んでいるかもしれません」**という余白を残した表現へ統一する。[1]

## 2. 共通運用ルール

Season1 は「発見編」であり、視聴者に説明しすぎず、直感的に「あるある」と感じてもらうことを優先する。そのため、各フォームも回答時間30秒以内を想定し、**1フォーム3問**、**回答は3択のみ**、**結果表示は3段階**で統一する。[1] [2]

| 項目 | 共通ルール |
|---|---|
| フォーム数 | 1話につき1フォーム |
| 設問数 | 1フォーム3問 |
| 回答時間想定 | 30秒以内 |
| 回答形式 | **よくある / たまにある / ほとんどない** |
| 採点ルール | よくある = 2点 / たまにある = 1点 / ほとんどない = 0点 |
| 結果表示 | 0〜1点 = 潜伏レベル / 2〜4点 = 感染注意レベル / 5〜6点 = 職場ウイルス反応あり |
| 話数別の最後の一言 | 各EPごとに1文だけ変え、SNSで拾いたくなる余韻を残す |
| 表現制限 | 「ストレス診断」「メンタル診断」などの表現は使わない |
| 結果文体 | 断定せず「潜んでいるかもしれません」に留める |
| 必須表記 | EP番号・ウイルス名・型を残す |

## 3. 採点ルールの共通表記

各フォームでは、次の採点ルールを共通表示として用いる。

| 回答 | 点数 |
|---|---|
| よくある | 2点 |
| たまにある | 1点 |
| ほとんどない | 0点 |

また、結果表示は次の3段階で共通化する。

| 合計点 | 結果表示 |
|---|---|
| 0〜1点 | 潜伏レベル |
| 2〜4点 | 感染注意レベル |
| 5〜6点 | 職場ウイルス反応あり |

## EP001｜ため息ウイルス｜自分本位型

フォームタイトル：
**あなたの職場にも「ため息ウイルス」は潜んでいますか？30秒チェック**

| 項目 | 内容 |
|---|---|
| 質問1 | 職場で誰かのため息によって、空気が重くなることがありますか？ |
| 質問2 | 誰かの不機嫌を、周囲が先回りして気にすることがありますか？ |
| 質問3 | 理由が分からない沈黙や機嫌に、仕事の進め方が左右されることがありますか？ |

結果表示：

| 合計点 | 表示文 |
|---|---|
| 0〜1点 | 潜伏レベル |
| 2〜4点 | 感染注意レベル |
| 5〜6点 | 職場ウイルス反応あり |

推奨表示：
**あなたの職場にも、ため息ウイルスが潜んでいるかもしれません。空気が重くなる朝には、もう反応が出ているのかもしれません。**

動画概要欄に貼る誘導文：
**あなたの職場にも「ため息ウイルス」はありますか？ 3問だけの職場あるあるチェックで確認してみてください。**

## 4. Season2フォーム拡張方針

Season2では、Season1の軽さを壊さない範囲で、各ウイルスの**株ごとの傾向**を把握するため、フォームを3問から**5問程度**へ拡張する。Season1が「まず参加してもらう」ための入口であるのに対し、Season2は「同じウイルスでもどの株が出やすいか」を、視聴者参加型で少しだけ深く観測する段階として位置づける。[1] [2]

Season2で追加するのは、原則として**株の表示**と**任意の職場規模項目**の2点のみとする。年代、性別、地域などの属性情報はSeason2では取得せず、まずは参加率と回答しやすさを優先する。IPによる地域推定も使わず、引き続き**創作コンテンツ連動の職場あるあるチェック**として運用する。

| 項目 | Season1 | Season2 |
|---|---|---|
| 設問数 | 3問 | 5問程度 |
| 株 | 非表示 | 表示 |
| 職場規模 | なし | 任意 |
| 年代 | 取得しない | 取得しない |
| 性別 | 取得しない | 取得しない |
| 地域 | 取得しない | 取得しない |
| IP地域推定 | 使わない | 使わない |

### Season2 職場規模項目

質問：
**あなたの職場規模を教えてください（任意）**

選択肢：

| 選択肢 |
|---|
| 1〜10人 |
| 11〜50人 |
| 51〜100人 |
| 101〜500人 |
| 501人以上 |
| 回答しない |

### Season2の目的

Season2は「変異編」として、同じウイルスでも**どのような株が出やすいか**を視聴者参加型で把握する段階とする。ただし、データ収集を前面に出しすぎず、あくまで**創作コンテンツ連動の職場あるあるチェック**としての軽さを維持する。

また、Season全体のフォーム設計は次の段階目標で整理する。

| Season | 目的 |
|---|---|
| Season1 | 気軽に参加してもらう |
| Season2 | 株ごとの傾向を見る |
| Season3 | ワクチン・防衛策へつなげる |
| Season4 | 職場ウイルス分布図へ展開する |

## 5. 運用上の注意

フォーム実装時は、タイトル、結果表示、概要欄文のどこでも**診断の断定性を上げすぎない**ことが重要である。特に「あなたは感染しています」「職場が危険です」などの断定表現は避け、必ず**創作コンテンツ連動の職場あるあるチェック**として扱う。[1] [2]

また、将来これらを「職場ウイルス度チェック」本体へ統合する場合でも、EP番号、ウイルス名、型の表記は残し、単発企画ではなくシリーズ資産として再利用できる形を維持する。
