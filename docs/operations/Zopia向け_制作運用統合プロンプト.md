# Zopia向け 制作運用統合プロンプト

## 1. 文書の目的

本書は、これまで別々に存在していた **Zopia向けの2レーン運用指示**、**レギュラー3名の修正プロンプト案**、**既存ロゴと正式タイトルの利用指示** を、実制作でそのまま使いやすい形へ統合した運用文書である。目的は、今後の『職場ウイルス図鑑』制作において、**実写版とアニメ版の役割分離**、**シリーズ共通要素の固定**、**レギュラーキャラクターの安定運用** を一つの文書で管理できるようにすることにある。[1] [2]

本書では、Zopia側へ最初に伝えるべき全体方針から、ロゴ・タイトルの固定ルール、レギュラー3名の補正方針、アニメ版再調整プロンプト、さらに実務上の投入順までを一体で整理する。これにより、従来のように複数のメモを行き来せずとも、**1つの起点文書**から制作指示を出せる状態を目指す。

## 2. 統合方針の要約

今回の統合では、3文書の役割を次のように再編した。まず、**2レーン運用の定義**を最上位ルールとして置く。次に、その共通ルールの下で、**ロゴと正式タイトルは共通固定資産**として扱う。さらに、**守屋守・相沢律子・クマ課長** については、実写版とアニメ版で別人化しないよう、年齢感、圧、疲労感、服装ニュアンスを統一方針として持たせる。[1] [2]

| 統合対象 | この文書での扱い |
|---|---|
| アニメ実写2レーン運用 | 全体ルールとして冒頭に統合 |
| レギュラー3名修正 | キャラ別補正方針として統合 |
| 既存ロゴ・タイトル利用 | 共通ブランド要素として統合 |

## 3. まずZopiaへ伝える全体方針

最初に、今後の制作方針そのものを明確に固定する。とくに重要なのは、**1EPを共通脚本の起点にしつつ、実写版とアニメ版は別プロジェクトとして扱う** こと、そして **ロゴとタイトルだけは両レーンで共通利用する** ことである。

### 全体方針伝達プロンプト

```text
今後の『職場ウイルス図鑑』は、各エピソードを「実写版」と「アニメ版」の2レーンで並行制作します。

重要ルール:
- 1EP = 1共通脚本 + 2レーン（実写版 / アニメ版）で運用する
- 実写版とアニメ版は別プロジェクトとして扱う
- 実写版は realistic live-action / realistic_3d_cg 系で統一する
- アニメ版は anime / stylized office short drama 系で統一する
- 実写版の画像資産をそのままアニメ化するのではなく、アニメ版はアニメ版として同一人物を再設計する
- ただし、キャラクターの役割、年齢感、空気感、関係性は実写版と揃える
- 実写版とアニメ版で別人化しないこと
- 今後の動画化も、実写版とアニメ版を必ず別々に進める
- 既存のシリーズロゴ「職場ウイルス図鑑」は継続利用する
- 各エピソードの正式タイトルは省略・意訳せず、そのまま使う

今回の依頼では、必要に応じてアニメ版キャラクターを、すでに良好な方向性が出ている実写版の雰囲気に寄せて再調整してください。
```

## 4. 共通ブランド要素の固定

『職場ウイルス図鑑』では、キャラクターや背景スタイルはレーンごとに分けても、**シリーズロゴと各話の正式タイトルは共通固定資産**として維持するほうが、IPとしての一貫性が高い。そのため、ロゴの別デザイン化や、タイトルの省略・言い換えは避けるべきである。

| 要素 | 方針 |
|---|---|
| シリーズロゴ | 既存ロゴ「職場ウイルス図鑑」を継続利用する |
| エピソードタイトル | 既存の正式タイトルをそのまま使用する |
| レーン差分 | キャラクターと背景スタイルのみ分ける |
| 禁止事項 | ロゴの別デザイン化、タイトルの省略・意訳・言い換え |

### ロゴ・タイトル固定プロンプト

```text
ロゴとタイトルの扱いは以下で固定してください。
- 既存ロゴ「職場ウイルス図鑑」を継続利用する
- 各エピソードのタイトルは正式名称をそのまま使用する
- アニメ版・実写版ともに同じロゴと正式タイトルを使う
- タイトルの言い換えやロゴの別デザイン化はしない
- ロゴやタイトルは、オープニング、タイトルカード、命名表示、サムネイル相当の静止画で使用してよい
```

### EPごとに差し替える汎用版

```text
今後の『職場ウイルス図鑑』制作では、既存ロゴ「職場ウイルス図鑑」を継続利用してください。
今回の対象エピソードは EP{{話数}}「{{正式タイトル}}」です。

ルール:
- 既存ロゴをそのまま利用する
- エピソードタイトルは正式名称をそのまま表示する
- アニメ版・実写版ともに同じロゴと正式タイトルを使う
- タイトルの省略、意訳、言い換えはしない
- ロゴだけ差し替えたり、新しい別ロゴを作らない
```

## 5. レギュラー3名の共通補正方針

Season 1前半の固定レギュラーは、**守屋守・相沢律子・クマ課長** の3名を起点に安定化させる。派手なアニメ化や極端な美化ではなく、**日本の職場ショートドラマ寄りのリアルベース** を保ちながら、シリーズで継続利用できる基準絵へ寄せることが重要である。[1] [2]

> Season 1前半の固定レギュラー3名として描写する。同一人物として継続利用できるアニメ調キャラクターデザイン。日本の一般的なオフィスが舞台。リアル寄りのショートドラマトーン。過度に若くしすぎない。過度に美化しすぎない。派手な漫画デフォルメは避ける。全身が見える正面立ち姿。背景はシンプルな無地またはオフィス用グレー。服装と顔立ちはシリーズで継続利用できる安定感を優先する。

| キャラクター | 補正の核 | 注意点 |
|---|---|---|
| 守屋守 | 若すぎる印象を抑え、疲労感ある中堅社員へ寄せる | 爽やかすぎる若手化を避ける |
| 相沢律子 | 知性と静けさを強め、補強者の空気を明確にする | 優しすぎる人物像に寄せすぎない |
| クマ課長 | 怒鳴らず圧を出す管理職として重さを足す | 露骨な悪役顔にしない |

### 守屋守 用プロンプト

```text
Season 1前半の固定レギュラー「守屋守」。30代後半前後の日本人男性会社員。主人公、観測者、受け手の中心人物。真面目で誠実、現場寄り、少し疲れた雰囲気。若すぎず、整いすぎず、現実の職場にいそうな自然な顔立ち。目元にわずかな疲労感、感情を飲み込む静かな表情。清潔感のある黒髪短髪、作り込みすぎない実務的な印象。白シャツ系ビジネスカジュアル寄り、必要ならネイビー系ジャケットを合わせるが堅すぎない。細身すぎず、一般的な中堅会社員体型。日本のオフィスショートドラマ風、リアル寄りアニメ調、全身立ち姿、シンプル背景。
```

### 相沢律子 用プロンプト

```text
Season 1前半の固定レギュラー「相沢律子」。30代後半から40代前半の日本人女性会社員。静かな補強者、正当化側、空気を固める人物。知的で上品、落ち着いていて有能そうに見える。感情を出しすぎず、柔らかい口調でも職場の圧を静かに補強するタイプ。若すぎる印象にはしない。整った目元、穏やかながら少し距離感のある表情。きちんとしたまとめ髪または上品なオフィス向けヘアスタイル。落ち着いたグレーやダークトーンのオフィス服、知的で品のある会社員スタイル。日本のオフィスショートドラマ風、リアル寄りアニメ調、全身立ち姿、シンプル背景。
```

### クマ課長 用プロンプト

```text
Season 1前半の固定レギュラー「クマ課長」。50代前後の日本人男性管理職。圧の発生源、管理職側の象徴、構造圧の起点となる人物。大柄で肩幅が広く、スーツ姿が自然に似合う。怒鳴るタイプではなく、沈黙、ため息、短い指示だけで場を支配する存在感。悪人顔ではないが、逆らいづらい重い空気を持つ。少し厳しめの目元、重心の低い立ち姿、落ち着いた口数の少なそうな表情。髪は整った短髪で、一部白髪混じり。ダークグレーまたは落ち着いた色のスーツ。日本のオフィスショートドラマ風、リアル寄りアニメ調、全身立ち姿、シンプル背景。
```

## 6. アニメ版を実写版の雰囲気へ寄せる再調整

アニメ版は、実写版の顔をそのままコピーするのではなく、**実写版で成立した年齢感、疲労感、静かな圧、人間関係の温度** をアニメ調に翻訳するほうが安定する。したがって、修正方針は「実写に似せる」ではなく、**実写版の雰囲気をアニメ版へ移植する** という考え方で運用する。[1] [2]

| 項目 | 修正方針 |
|---|---|
| 守屋守 | 若すぎず、疲労感のある中堅社員へ寄せる |
| 相沢律子 | 知的で落ち着いた、静かな補強者の空気を強める |
| クマ課長 | 怒鳴らず圧を出す管理職として重さを足す |
| 全体トーン | アニメ調でもリアル寄り、日本の職場ドラマの空気を保つ |
| 禁止事項 | 過度な若返り、過度な美化、デフォルメ過剰、漫画的悪役化 |

### アニメ版再調整 本体プロンプト

```text
『職場ウイルス図鑑』EP001 ため息ウイルスのアニメ版キャラクターを再調整してください。

前提:
- 今後は実写版とアニメ版を別レーンで並行制作する
- 今回はアニメ版のみを調整対象とする
- アニメ版のスタイルは anime / stylized office short drama
- ただし雰囲気は、すでに生成された実写版の空気感に寄せる

重要な修正方針:
- アニメ風の記号化を強くしすぎない
- 若すぎるキャラデザインにしない
- 過度に美化しすぎない
- 現実の日本の会社にいそうな人物として描く
- 実写版で成立した「静かな圧」「疲労感」「日常に混ざる不穏さ」をアニメ調で再現する
- 実写版の顔をそのままコピーするのではなく、役割・年齢感・空気感をアニメ化する

キャラクター指定:
1. 守屋守
30代後半前後の日本人男性会社員。主人公、観測者、受け手の中心人物。真面目で誠実、少し疲弊気味。白シャツ系ビジネスカジュアル寄り。若すぎず、爽やかすぎず、静かに消耗している中堅社員。目元にわずかな疲労感、感情を飲み込む表情。

2. 相沢律子
30代後半から40代前半の日本人女性会社員。知的で上品、落ち着いたオフィス服。静かな補強者、正当化側、空気を固める人物。優しすぎる人物ではなく、柔らかいが理性的で、感情を出しすぎない。実写版の静かな存在感をアニメ調に置き換える。

3. クマ課長
50代前後の日本人男性管理職。大柄、スーツ姿、圧の発生源。怒鳴るのではなく、沈黙、ため息、短い指示だけで場を支配する。悪人顔にはしないが、逆らいづらい重さを持つ。実写版の威圧感と管理職感をアニメ調で再現する。

全体演出:
- 日本の一般的なオフィスショートドラマ風
- リアル寄りアニメ調
- 全身または半身で、シリーズ継続利用しやすい基準キャラとして描く
- 背景はシンプルなオフィスまたは無地
- 3名は同一シリーズの固定レギュラーとして見えるよう統一感を持たせる

禁止事項:
- 実写版と別人に見えるほど若返らせない
- 漫画的な誇張や派手なヒーロー感を出さない
- クマ課長を露骨な悪役顔にしない
- 相沢律子を可愛らしさ優先にしすぎない
- 守屋守を若手新人のようにしない
```

### 短縮補正プロンプト

```text
アニメ版キャラクターを、実写版の雰囲気に寄せて微調整してください。

- 守屋守は若すぎる印象を抑え、30代後半前後の疲労感ある中堅社員にする
- 相沢律子は知的で上品、感情を出しすぎない静かな補強者にする
- クマ課長は怒鳴らず圧を出す50代管理職として、体格と重さを少し強める
- 全体をリアル寄りアニメ調にして、日本の職場ドラマの空気に寄せる
- 過度な美化、若返り、デフォルメは避ける
```

## 7. 実務での使い分け

運用を安定させるには、すべての文を一度に投げ込むのではなく、**全体固定 → ブランド固定 → キャラ固定 → 再調整** の順で投入するほうがよい。これにより、Zopia側でルールの優先順位がぶれにくくなる。

| 手順 | 入れる文面 | 目的 |
|---|---|---|
| 1 | 全体方針伝達プロンプト | 2レーン運用と共通ルールを固定する |
| 2 | ロゴ・タイトル固定プロンプト | 共通ブランド要素を固定する |
| 3 | 各キャラクター用プロンプト | 基準絵を安定化させる |
| 4 | アニメ版再調整 本体プロンプト | 実写版の雰囲気をアニメ側へ移植する |
| 5 | 短縮補正プロンプト | 効きすぎた部分だけを微調整する |

また、初回生成時は長めの文面を使い、再修正時は短い補正指示へ切り替えるほうが、既存の良い要素を壊しにくい。

## 8. 運用上の注意

今回の統合方針では、レーンの違いは認めつつも、**IPとしての同一性は崩さない** ことを重視する。そのため、アニメ版と実写版で別人化するような若返り、過度な美化、キャラクター役割の変質は避けるべきである。また、ロゴと正式タイトルはシリーズの認知資産であるため、軽率な変更はしない。[1] [2]

| 観点 | 注意点 |
|---|---|
| レーン運用 | 実写版とアニメ版を混ぜない |
| キャラクター | 同一人物として継続利用できることを優先する |
| ロゴ | 既存ロゴを維持する |
| タイトル | 正式名称を省略・意訳しない |
| アニメ修正 | 実写の顔コピーではなく、雰囲気移植として扱う |
