# 職場ウイルス図鑑 - SNS Operation Policy

本ドキュメントは、「職場ウイルス図鑑」プロジェクトにおけるSNS（YouTube Shorts, TikTok, Instagram Reels等）の運用方針を定めたものです。

## 1. サイトレス・パンデミック戦略の実行
本プロジェクトは初期段階において公式サイトを持たず、SNS、Googleフォーム、Kindle出版などを独立した「感染経路」として扱う「サイトレス・パンデミック戦略」を採用します。
各媒体を起点としてユーザーが「職場ウイルス図鑑という世界観」に触れ、他の媒体へ回遊するような相互導線を設計します。

## 2. プラットフォーム別の運用方針
ショート動画を中心に、各プラットフォームの特性に合わせた運用を行います。

### YouTube Shorts
- **役割**: 幅広い層へのリーチと、チャンネル登録による継続的な視聴の獲得。
- **投稿内容**: 本編のショート動画。タイトルや説明文には検索されやすいキーワード（「あるある」「ブラック企業」「上司」など）を含めます。
- **導線**: コメント欄の固定メッセージや概要欄から、Googleフォーム（感染度チェック）やKindle（感染速報版）へのリンクを設置します。

### TikTok
- **役割**: 若年層を中心としたバイラルな拡散と、トレンドへの便乗。
- **投稿内容**: 本編のショート動画に加え、流行の音源やフォーマットを取り入れたスピンオフ的な短い動画も検討します。
- **導線**: プロフィールリンク（Linktree等）に各コンテンツへのリンクを集約し、動画内やキャプションでプロフィールへの遷移を促します。

### Instagram Reels
- **役割**: 視覚的な訴求と、ハッシュタグを通じた特定の関心層へのリーチ。
- **投稿内容**: 本編のショート動画。リールズの雰囲気に合わせ、カバー画像（サムネイル）のテキスト配置やデザインに工夫を凝らします。
- **導線**: プロフィールリンク（Linktree等）への誘導に加え、ストーリーズを活用してGoogleフォームのリンクを直接共有したり、Kindle出版の告知を行ったりします。

## 3. 投稿コンテンツの構成とトーン＆マナー
- **余韻の重視**: ショート動画は「違和感の発見と命名」までに留め、解決策や詳細な分類は出しすぎないようにします。動画の最後は「○○ウイルス」という名称を提示して終わり、「続きが見たくなる」余韻を残します。
- **共感の喚起**: 視聴者が「うちの職場にもいる！」「わかる！」と共感できるような、日常的な職場の風景を切り取ります。
- **ハッシュタグ戦略**: #職場あるある #ブラック企業 #上司の愚痴 #職場ウイルス図鑑 などの共通ハッシュタグに加え、エピソード固有のキーワード（#丸投げ #責任転嫁 など）を使用します。

## 4. 相互導線と受け皿の管理
公式サイトの代わりとして、以下のツールを簡易的な受け皿として活用し、相互導線を管理します。

- **Linktree等のリンクまとめサービス**: 各SNSのプロフィールに設置し、YouTube、TikTok、Instagram、Kindle、Googleフォームへのリンクを一元化します。
- **note固定記事**: 世界観の解説やエピソード一覧、各媒体へのリンクをまとめた記事を作成し、必要に応じて誘導先とします。
- **Kindle著者ページ**: 出版物の一覧と、SNSやフォームへのリンクを掲載します。

## 5. ユーザーとのコミュニケーション
- コメント欄では、視聴者の共感や体験談を促すような問いかけ（「あなたの職場にもこんな人いませんか？」など）を行います。
- 全てのコメントに返信する必要はありませんが、議論が盛り上がっているコメントや、面白い体験談には「いいね」や短い返信を行い、コミュニティの活性化を図ります。
