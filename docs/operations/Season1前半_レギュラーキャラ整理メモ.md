# Season 1前半 レギュラーキャラ整理メモ

## 1. このメモの目的

本メモは、**『職場ウイルス図鑑』Season 1前半（EP001〜EP005）におけるレギュラー3名の固定条件**を、2プロジェクト同時生成運用に対応した形で整理するための基準文書である。前半5話では、**守屋守・相沢律子・クマ課長**を基本レギュラーとして固定し、EP001で成立した役割・基本外見・関係性を、アニメ版と実写版の両方へ引き継ぐ。[1] [2]

この整理の狙いは、毎話ごとに人物認識をやり直さなくてもよいシリーズ設計を作ることにある。すでに前半5話の配置整理では、**守屋守を視点軸、クマ課長を象徴的圧力源、相沢律子を静かな同調・補強役**として揃える方針が確認されており、本メモはその判断をキャラクター運用の観点から明文化する。[3]

## 2. 基本原則

Season 1前半で固定するのは、**誰が出るか、どういう役割を持つか、どのような基本外見で見せるか**である。一方で、各話のウイルス構造に応じて、**表情、ポーズ、演技、構図、感情強度**は柔軟に変えてよい。すなわち、キャラクターを硬直化させるのではなく、**同一人物の差分運用**として広げていくことを前提にする。[1] [5]

| 項目 | 固定する内容 | 可変にしてよい内容 |
|---|---|---|
| 人物識別 | 名前、役割、基本関係性 | その回での登場比重、出番の長さ |
| 外見 | 年齢感、髪型、顔立ち、服装系統、体格 | 表情、視線、ポーズ、カメラ距離 |
| 演技 | キャラの立ち位置、圧の出し方の方向性 | 反応の強さ、間、沈黙、言い回しの細部 |
| 資産管理 | 既存キャラとして管理すること | 新規差分の追加自体 |

## 3. レギュラー3名の固定定義

レギュラー3名は、単に登場人物を固定するためではなく、シリーズの見え方を固定するために置いている。**守屋守が違和感を受け取り、相沢律子がそれを静かに正当化・補強し、クマ課長が圧の起点になる**という三角構造が、前半5話の観測軸である。[1] [3]

| キャラクター | シリーズ内の基本位置付け | 引き継ぐ基本人物像 | 引き継ぐ外見の軸 | 変化させてよい要素 |
|---|---|---|---|---|
| **守屋守** | 主人公 / 観測者 / 受け手の中心人物 | 真面目、誠実、現場寄り、少し疲弊気味、中堅社員 | 30代後半前後、日本人男性、白シャツ系ビジネスカジュアル寄り、EP001基準の髪型・顔立ち | 視線停止、飲み込み、遅れて気づく表情、疲労の度合い、構図上の追い詰められ方 |
| **相沢律子** | 静かな補強者 / 正当化側 / 空気を固める人物 | 有能そう、落ち着いている、感情を出しすぎない、政治的・調整的 | 30代後半〜40代前半、日本人女性、知的で上品、落ち着いたオフィス服、EP001基準の髪型・顔立ち | 無言同調、やわらかい正当化、依頼の微笑、理性的補強、視線やうなずきの強さ |
| **クマ課長** | 圧の発生源 / 管理職側の象徴 / 構造圧の起点 | 大柄、威圧感、管理職感、悪人ではなく圧の構造を体現する人物 | 50代前後、日本人男性、大柄、スーツ系、EP001基準の髪型・顔立ち | ため息、沈黙、確認圧、口頭圧、遠景圧など圧の出し方の違い |

## 4. 資産管理上の扱い

キャラクターの継続性は、脚本上の役割だけでなく、**命名規則と差分運用の一貫性**によって支えられる。正式保管庫は `assets/reference/` であり、ここには承認済みの基準画像と最小限の有効差分のみを置く。実際のZopia手動投入は `assets/zopia_upload/season1_ep001_ep005/` を使うが、ここはあくまで出荷箱であって、正本ではない。[4] [6]

また、前半5話で増える表情差分や補助画像は、**新キャラではなく既存キャラ差分**として管理する。したがって、守屋守の観測系差分、相沢律子の補強系差分、クマ課長の圧系差分は、いずれも同一キャラの延長として登録し、別人物として枝分かれさせない。[1] [5]

| 管理対象 | 正本 | 運用上の扱い |
|---|---|---|
| 基準顔・基準立ち姿 | `assets/reference/characters/` | キャラクター同一性の起点として維持する |
| 実写版投入用コピー | `assets/zopia_upload/season1_ep001_ep005/` | 手動アップロードのために必要分だけ並べる |
| 表情差分 | `expression` 系命名で既存キャラ差分として管理 | 別キャラ扱いしない |
| 姿勢差分 | `pose` 系命名で補助的に管理 | 同一人物の見せ方の幅として扱う |

## 5. 2プロジェクト運用との関係

2プロジェクト同時生成運用では、**1EP = 1共通脚本 + 2レーン（アニメ版 / 実写版）** が前提になっている。そのため、キャラクター定義もレーンごとに別人化させるのではなく、**共通脚本の中にある同一人物を、アニメ版と実写版で異なる方法で再現する**という考え方で揃える必要がある。[2]

アニメ版では、共通キャラ定義文を毎話で使い回すことで別人化を防ぐ。実写版では、EP001基準画像を主参照資産として継続投入し、顔立ち・髪型・年齢感・服装系統のぶれを抑える。どちらのレーンでも、変えてよいのは演技と演出であり、**キャラの核そのものではない**。[1] [2]

## 6. 運用上の判断基準

前半5話の制作判断に迷った場合は、まず**「この変更は同一人物の差分として説明できるか」**を確認する。説明できるなら採用余地があり、説明できないなら、別人化または役割逸脱の可能性が高い。これは、シリーズ初期の視聴者認知と制作資産の蓄積を両立させるための最小基準である。[1]

| 判断場面 | 採用してよい例 | 避けるべき例 |
|---|---|---|
| 守屋守の変化 | 疲労感の増減、視線停止、遅れて気づく表情 | 別人級に若返る、過度に反抗的になる |
| 相沢律子の変化 | 微笑、視線落とし、理性的補強、丁寧な依頼 | 露骨な悪役化、感情過多な大演技 |
| クマ課長の変化 | ため息、沈黙、軽口指示、会議支配 | 感情的怒号主体の別キャラ化 |
| 新規素材追加 | 既存キャラ差分として命名・登録 | 新キャラ扱いでフォルダを増やす |
