# EP001 ラフ編集 組み立て手順

## 目的

本書は、`docs/operations/EP001_試験編集版_実制作パッケージ.md` を、実際のラフ編集作業へそのまま接続できる**タイムライン組み立て用の実務手順書**へ落とし込んだものである。目的は、EP001の30秒試験編集版について、背景・人物・UI・字幕・音の配置順を固定し、編集者が迷わず**最初のラフ版を一度通せる状態**をつくることにある。

今回のラフ編集では、完成度を先に追うのではなく、**秒感、空気停止、視線誘導、命名締めの効き**を確認することを優先する。そのため、すでに追加済みの優先差分4点は積極的に使い、未追加の3点は既存差分で近似しながら先に通しを作る。

## ラフ編集の固定前提

EP001は、クマ課長のため息そのものより、**守屋守が場の異変を観測してしまう瞬間**を中心に組み立てる。したがって、カットの強さは大げさな芝居ではなく、短い停止、音の引き、視線の流れ、無言同調で成立させる。

| 項目 | 固定値 | 実務上の扱い |
|---|---|---|
| 尺 | 29〜31秒 | 30秒前後で組み、命名締めは削らない |
| 画面比率 | 9:16 | 1080×1920基準 |
| ラフ版の優先課題 | 秒感確認 | 完パケ品質より通し成立を優先 |
| 主視点 | 守屋守 | 守屋守の停止と観測を軸にする |
| ロゴ | 常設 | 右上固定、画面幅12%前後 |
| 字幕 | 少なめ | Shot 2・4・7のみに絞る |
| 命名締め | 必須 | `ai_log_overlay_v01.png` を使用 |

## レイヤー順とコンポ構成

ラフ版でも、レイヤー順だけは本番想定で固定しておく方が差し替えが速い。とくにロゴと命名UIの位置関係は早い段階で固定しておく。

| レイヤー | 内容 | 備考 |
|---|---|---|
| L1 | 背景 | 背景4枚をショットごとに差し替える |
| L2 | 人物ベース | 基準素材またはポーズ素材 |
| L3 | 人物差分 | 追加済み優先差分4点を優先使用 |
| L4 | 小物補助 | 必要最低限。ラフでは省略可 |
| L5 | 字幕 | Shot 2, 4, 7 中心 |
| L6 | ロゴ | 右上固定 |
| L7 | 命名UI | Shot 7のみ |

## 作業順序

ラフ編集は、まず通しを完成させ、その後で差分や見た目を詰める。したがって、最初から細部を作り込まず、次の順で進めると安定する。

| 手順 | やること | 完了条件 |
|---|---|---|
| 1 | 背景4枚、ロゴ、命名UIを読み込む | 全ショットの背景とUIの受け皿ができる |
| 2 | 3名の基準素材と追加済み差分4点を読み込む | 人物差し替えが可能になる |
| 3 | Shot 1〜7を静止画ベースで仮配置する | 30秒前後の通しが成立する |
| 4 | ため息SE、環境音、命名前の音引きを入れる | 空気停止の感触が確認できる |
| 5 | Shot 2・4・7の字幕候補だけ仮置きする | 字幕量の過不足が確認できる |
| 6 | ロゴ位置、暗転秒感、命名締めを微調整する | 試験版として確認可能になる |

## ラフ版で確認すべき判定項目

この段階で確認すべきなのは、素材点数の多さではなく、**ウイルス命名として怖さが成立するか**である。以下の観点で一度通して確認する。

| 判定項目 | 見るポイント |
|---|---|
| ため息の効き | Shot 3で空気が止まったと感じるか |
| 守屋守の観測 | Shot 4で「見てしまった感」が出るか |
| 相沢律子の補強 | Shot 5で無言同調が怖さを増しているか |
| 再開の不気味さ | Shot 6で誰も触れずに戻る感じが出るか |
| 命名締め | Shot 7で説明ではなく観測結論に見えるか |
| 字幕量 | 少ないままで意味が伝わるか |

## 現時点の次アクション

本書を使えば、EP001ラフ編集はそのまま着手できる。最優先は、**まず一度30秒の通しを作ること**である。その後、未追加の残差分3点を補完し、Shot 4、5、7の精度を上げれば、本番確認版へ自然に接続できる。
