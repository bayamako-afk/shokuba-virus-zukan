# Season 1前半 EP001〜EP005｜必要素材一覧

## 目的
本書は、EP001〜EP005の30秒版shotlist初稿を、**実写コンテ化と素材発注**に直結する単位へ分解した一覧である。目的は、各話ごとに必要な素材を都度考えるのではなく、**共通で先に押さえるべき素材**と、**各話固有で追加すべき素材**を切り分けることにある。

## 運用方針
Season 1前半は、守屋守・クマ課長・相沢律子の3名を固定しているため、人物素材も**単発の役ごとの作り捨て**ではなく、**基準顔 + 表情差分 + 姿勢差分**として揃える方が効率がよい。背景も、同一職場感を維持するために、朝・昼・夕の差し替えで回せる共通環境を優先する。

## 素材カテゴリ定義

| カテゴリ | 内容 | 使い方 |
|---|---|---|
| 人物基準 | 3人の基準ビジュアル、服装、髪、年齢感、役職感 | 全話共通の土台 |
| 表情差分 | 顔の微差分、圧、疲労、補強表情など | 実写コンテ・image2差分作成用 |
| 背景 | 執務室、会議室、廊下、机周りなど | 同じ会社感の維持 |
| UI / 画面 | チャット、観測ログ、図鑑カード、PC画面など | 情報提示・命名演出 |
| 小物 / 手元 | 書類、付箋、会議メモ、PC、スマホなど | 異変の可視化 |
| 音・演出補助 | ため息、タイピング、沈黙、暗転など | 不気味さの補強 |

## 先に作る順番
素材制作の順番は、**人物基準 → 表情差分 → 背景 → UI → 小物**が最も効率的である。とくに前半5話は人物差分の再利用率が高いため、背景や小物より先に、3人の基準顔と横断差分を固めた方が、のちの実写コンテ化が安定する。
