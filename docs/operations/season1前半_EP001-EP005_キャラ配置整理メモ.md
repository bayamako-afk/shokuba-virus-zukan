# Season 1前半 EP001〜EP005 キャラ配置整理メモ

## 目的
本メモは、Season 1前半のEP001〜EP005を**同じ職場・同じ主要キャラ基準**で見せるために行った最小修正の整理用メモである。今回の調整では、各話を全面改稿せず、既存の骨格・尺・トーンを維持したまま、**守屋守 / クマ課長 / 相沢律子**の見え方だけを揃えた。

## 基本方針
Season 1前半は、視聴者に『職場ウイルス図鑑』の世界観と人物関係を覚えてもらう期間として扱う。そのため、EP001〜EP005では、**守屋守を視点軸、クマ課長を象徴的圧力源、相沢律子を静かな同調・補強役**として固定し、各話ごとに出番の濃淡だけを調整する方針に寄せた。

| 話数 | 主視点 | 圧力源 | 静かな補強役 | 今回の寄せ方 |
|---|---|---|---|---|
| EP001 ため息ウイルス | 守屋守 | クマ課長 | 相沢律子 | 既存確定版を維持。3人の関係性が最も見えやすい導入回として扱う。 |
| EP002 差し戻し確認ウイルス | 守屋守 | クマ課長 | 相沢律子 | 差し戻しの発生源をクマ課長に寄せ、相沢律子は確認のやり直しを静かに自然化する立ち位置へ調整。 |
| EP003 責任転嫁ウイルス | 守屋守 | クマ課長 | 相沢律子 | もともとの被感染者視点寄り設計を壊さず、責任が守屋守へ流れる会議構図に置き換えて前半の人物軸を固定。 |
| EP004 口頭指示ウイルス | 守屋守 | クマ課長 | 相沢律子 | 口頭指示の起点をクマ課長に固定し、相沢律子は止めない・流すことで空気を補強する配置へ寄せた。 |
| EP005 なぜか君担当ウイルス | 守屋守 | クマ課長（構造圧） | 相沢律子 | もともとの他観測者回を、守屋守に仕事が偏る構図へ最小変更。相沢律子の感じの良い依頼とクマ課長の断りづらい空気で偏りを成立させる。 |

## 話ごとの補足
EP001は、すでに前提となる3人の関係が整っていたため、今回の方針変更に対する追加修正は不要と判断した。導入回として、守屋守が異常を観測し、クマ課長が空気を止め、相沢律子が無言で場を補強する構図がそのまま前半全体の基準になる。

EP002は、守屋守の視点を維持したまま、差し戻しの発生源をクマ課長へ明示した。相沢律子は前面に出すのではなく、「もう一度確認した方がいいですね」と空気を整える側に置くことで、露骨な敵役ではないが怖い存在として積み上げる形にした。

EP003は、変更幅がもっとも大きい回だが、やったことは**主人公軸の固定**に限っている。会議構図や責任転嫁の核はそのまま残し、守屋守が引き受け役にされる構図へ寄せたうえで、クマ課長が方向づけ、相沢律子が理性的な言葉で補強する形に整理した。

EP004は、守屋守が口頭指示に引っかかる流れを維持しつつ、指示の起点をクマ課長に固定した。相沢律子は口頭指示を止めず、そのまま流す役割にすることで、「悪意ではないが異常を正常化する人」としての連続性を持たせた。

EP005は、もともと他観測者回として設計されていたが、前半5話のキャラ固定方針を優先し、守屋守が頼みごとを抱え込む回へ最小変更した。相沢律子は感じの良い依頼者として、クマ課長は直接命じなくても断りづらい職場空気の起点として機能させ、同じ会社の同じ構造の中で起きている話だと分かるように寄せた。

## 反映対象
今回の修正では、主に**主人公表記、登場人物欄、相手側の位置づけ、周囲の補強役、導入と異変の起点**を整えた。構成、命名、尺、Season 1の「違和感の発見と命名」で止める方針は維持している。
