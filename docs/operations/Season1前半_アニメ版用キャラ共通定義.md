# Season 1前半 アニメ版用キャラ共通定義

本書は、**Season 1前半（EP001〜EP005）のアニメ版をZopia完結型で運用する際に、毎話使い回すための共通定義文**である。前半5話では、守屋守・相沢律子・クマ課長の3名を基本レギュラーとして固定し、EPごとに変えるのは主に症状、演技、カット構成、表情差分の強さに限る。[1] [2]

アニメ版は実写版のような先行素材投入を前提にしないため、**毎回の指示文の中で同一人物であることを明記すること**が、別人化防止の中心になる。本フォーマットは、既存のGeminiCL標準フォーマットの考え方を踏まえつつ、アニメ版用に簡潔化した再利用テンプレートとして用いる。[1] [3]

## 1. 新規プロジェクト作成

- [ ] プロジェクト名: `職場ウイルス図鑑 EP00X {各話タイトル} アニメ版`
- [ ] 言語設定: 日本語
- [ ] アスペクト比: 9:16（ショート動画形式）
- [ ] スタイル: anime / stylized office short drama / Japanese office setting
- [ ] 尺の前提: 30秒想定
- [ ] レーン前提: **アニメ版はZopia完結型**であり、実写版の基準画像投入前提を持ち込まない
- [ ] シリーズ前提: **Season 1前半の固定レギュラー3名を継続利用する**

## 2. 脚本最小構成

- [ ] 世界観: 「日本の一般的なオフィスで起きる、説明しすぎない職場ショートドラマ」
- [ ] 共通構図: 「守屋守が違和感を受け取り、クマ課長が圧の起点となり、相沢律子が静かに空気を補強する」
- [ ] 各話で変える要素: 「ウイルス名、場面、症状の出方、圧の種類、ラストの命名文」
- [ ] 各話で固定する要素: 「主要3名の名前、役割、基本外見、関係性」
- [ ] 文章量: 200〜300文字程度の簡潔なプロット
- [ ] 冒頭指示に必ず入れる文: 「Season 1前半の同一レギュラー3名を継続利用し、EP001の人物印象を維持する」

> 守屋守・相沢律子・クマ課長は、Season 1前半を通して継続する同一人物である。守屋守は現場側の観測者、相沢律子は静かな補強者、クマ課長は圧の発生源として固定し、EP001で成立した人物印象を維持したまま、今回のウイルス現象だけを変化させる。表情、ポーズ、演技、構図は各話に合わせて変えてよいが、名前、役割、基本外見、関係性は変えない。

## 3. キャラ設定

- [ ] 守屋守の名前を固定して記述する
- [ ] 守屋守の役割を「主人公 / 観測者 / 受け手の中心人物」と明記する
- [ ] 守屋守の外見を「30代後半前後の日本人男性会社員。真面目で誠実、少し疲れた雰囲気。白シャツ系ビジネスカジュアル寄り」と記述する
- [ ] 相沢律子の名前を固定して記述する
- [ ] 相沢律子の役割を「静かな補強者 / 正当化側 / 空気を固める人物」と明記する
- [ ] 相沢律子の外見を「30代後半〜40代前半の知的で上品な日本人女性社員。落ち着いたオフィス服。感情を出しすぎず、柔らかいが圧を補強する」と記述する
- [ ] クマ課長の名前を固定して記述する
- [ ] クマ課長の役割を「圧の発生源 / 管理職側の象徴 / 構造圧の起点」と明記する
- [ ] クマ課長の外見を「50代前後の大柄な日本人男性管理職。スーツ姿。怒鳴らず、沈黙やため息や短い指示で場を支配する」と記述する
- [ ] 同一人物指定を入れる: 「3名ともEP001から継続している同一人物として描写する」
- [ ] 許容変化を入れる: 「表情、ポーズ、目線、演技の強さ、デフォルメ度は各話に応じて変えてよい」
- [ ] 禁止事項を入れる: 「別人級の顔変化、役割逆転、主要3名の差し替えはしない」

| キャラクター | 固定役割 | 固定外見の軸 | 各話で動かしてよい点 |
|---|---|---|---|
| **守屋守** | 観測者 / 受け手 | 30代後半前後、真面目、疲弊気味、現場寄り | 停止、飲み込み、疲労、遅れて気づく反応 |
| **相沢律子** | 静かな補強者 / 正当化側 | 知的、落ち着いたオフィス女性、上品 | 視線落とし、微笑、理性的補強、軽いうなずき |
| **クマ課長** | 圧の発生源 / 管理職象徴 | 50代前後、大柄、スーツ、威圧感 | ため息、沈黙、確認圧、会議支配、口頭圧 |

## 4. コンテ化

- [ ] カット数: 5〜7カットを基準にする
- [ ] 画面構成: 「導入 → 異変 → 巻き込まれ → 周囲の平然 → 命名」の順で組む
- [ ] 主観軸: 守屋守の違和感を中心に見る
- [ ] 圧の見せ方: クマ課長は大げさな悪役化ではなく、日常に混ざる圧として見せる
- [ ] 補強の見せ方: 相沢律子は露骨な加害者ではなく、静かな正当化として見せる
- [ ] 演出トーン: 日本の職場ショートドラマ風、静かなじわ怖、説明しすぎない
- [ ] 最終カット: 命名表示または観測ログで締め、Season 1の図鑑感を維持する

## 5. 本実行前注意点

- [ ] 最初の指示文に「同一人物継続」「EP001印象維持」「役割固定」を必ず入れる
- [ ] 実写版のような写真参照前提を書き込まない
- [ ] キャラの核を壊す過剰デフォルメを避ける
- [ ] 大げさなホラー化や漫画的誇張だけに寄せすぎない
- [ ] 主要3名以外の新キャラは必要最小限のサブ扱いに留める
- [ ] 新規差分は既存キャラの差分として扱い、別キャラ扱いしない
- [ ] 各EPで文面を少し変える場合でも、名前・役割・基本外見の3点は削らない
