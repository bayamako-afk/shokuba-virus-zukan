# EP001 試験編集版 構成指示書

## 目的

本書は、**Season 1 前半ショート動画共通編集テンプレート**に沿って、EP001「ため息ウイルス」の**試験編集版を1本だけ先行作成するための実務指示書**である。今回の試験版の目的は、完成納品ではなく、以下の3点を視聴体感で検証することにある。

| 検証項目 | 今回確認したいこと |
|---|---|
| ロゴサイズ | 右上常設ロゴが邪魔に見えず、しかしシリーズ認知は維持できるか |
| 字幕量 | セリフや説明が多すぎず、空気の怖さを損なっていないか |
| 命名締め | 観測ログ型の締めが短すぎず長すぎず、余韻として機能するか |

## 試験編集版の前提

EP001は、前半5話の中でも構造が最もシンプルであり、**日常 → ため息 → 空気停止 → 無言同調 → 命名**の流れが明確である。そのため、共通編集テンプレートの初期検証には最も向いている。編集の狙いは、派手な見せ方を探すことではなく、**普通の職場のまま空気だけが少し冷える**感覚を成立させることである。

## 完成尺の目安

試験編集版の完成尺は、**29〜31秒**を推奨する。30秒ぴったりに合わせる必要はないが、命名締めの余韻を削ってまで秒数だけを厳密化しない。

| 項目 | 推奨値 |
|---|---|
| 画角 | 9:16 |
| 解像度 | 1080 × 1920 |
| 完成尺 | 29〜31秒 |
| カット数 | 7カット |
| 命名型 | 観測ログ型 |
| 主視点 | 守屋守 |

## タイムライン構成

EP001試験編集版は、既存shotlistをそのまま土台にしつつ、検証しやすいように**ロゴ確認区間**、**字幕密度確認区間**、**命名締め確認区間**の3つを明確に分けて考える。

| Block | 秒数目安 | 役割 | 主な確認点 |
|---|---:|---|---|
| A | 0〜8秒 | 日常導入 | ロゴが背景と人物に干渉しないか |
| B | 8〜21秒 | 異常の発火と伝播 | 字幕量が説明過多になっていないか |
| C | 21〜31秒 | 空気固定化と命名締め | 命名までの落ち方に余韻があるか |

## 字幕量の基準

今回の試験版では、字幕は**少なめを基準**にする。EP001は説明で見せる回ではなく、ため息一つで場が止まる感覚を読ませる回であるため、字幕を足しすぎると世界観が浅く見える。

| 区分 | 推奨 |
|---|---|
| 字幕ありショット | Shot 2, Shot 4, Shot 7 のみを基本候補とする |
| 字幕なし推奨ショット | Shot 1, 3, 5, 6 |
| 1カットあたりの行数 | 1行、必要でも2行まで |
| 1行の情報量 | 短文中心 |
| 避けたい字幕 | 「気まずい」「圧がある」など感情の説明文 |

字幕を入れる場合でも、守屋守の心情をすべて言語化しない。**視線停止と間で伝わるものは、字幕で上書きしない**ことが重要である。

## ロゴ検証ルール

今回の試験版では、ロゴは**常時表示のまま検証**する。ロゴを一時的に消す版は、必要であれば比較用の第二案として後で作ればよい。まずは標準運用がそのまま成立するかを見る。

| 項目 | 指示 |
|---|---|
| 使用ファイル | `assets/reference/brand/shokuba_virus_zukan_logo_main_v01.png` |
| 配置 | 右上固定 |
| サイズ | 画面幅の12%前後から試す |
| 不透明度 | 85%前後から試す |
| 優先確認ショット | Shot 1, Shot 5, Shot 7 |

とくにShot 5では、相沢律子の視線落としとロゴが競合しないかを確認する。Shot 7では、命名演出とロゴの主張がぶつからないかを確認する。

## 命名締めの検証ポイント

EP001の試験版で最も重要なのは、命名締めが**説明のオチ**ではなく、**観測の結論**として落ちるかである。したがって、終盤のテンポは少し遅めでよい。

| 項目 | 推奨 |
|---|---|
| 命名前の間 | 0.5〜1.0秒 |
| 暗転 | 短く入れて可 |
| 使用素材 | `ai_log_overlay_v1` |
| 命名表示時間 | 約1.5〜2.0秒 |
| 印象目標 | 「気づいてしまった」で終わる |

短すぎると情報処理で終わり、長すぎるとテンポが落ちる。初回試験では**1.8秒前後**を目安に置くと判定しやすい。

## 必要なシーン背景画像

はい、**EP001試験編集版では背景画像が必要**である。ただし、最小構成で進めるなら、新規に大量制作する必要はなく、以下の共通背景があれば成立する。

| 背景ID | 必須度 | 用途 | 対応ショット |
|---|---|---|---|
| `office_morning_common` | 最優先 | 朝の執務室全景 | Shot 1, 6 |
| `desk_row_common` | 最優先 | 守屋守席周辺 | Shot 2 |
| `manager_desk_common` | 最優先 | クマ課長席 | Shot 3 |
| `aizawa_desk_common` | 高 | 相沢律子席または近接業務席 | Shot 5 |

追加であると便利だが、試験版段階では必須ではない背景も整理しておく。

| 背景ID | 任意度 | 用途 |
|---|---|---|
| `office_morning_common_crop_close` | 任意 | Shot 6を別画に見せたい場合の切り出し用 |
| `moriya_reaction_bg_soft` | 任意 | Shot 4で守屋守反応の見え方を安定させたい場合 |

結論として、**最低4背景で試験版は制作可能**である。もし背景不足なら、最優先で必要なのは `office_morning_common`、`desk_row_common`、`manager_desk_common` の3点であり、`aizawa_desk_common` は既存全景からの切り出しでも代替可能である。

## 必要人物素材

背景だけでは成立しないため、人物差分も最低限これだけは必要になる。

| キャラ | 必要差分 | 用途 |
|---|---|---|
| 守屋守 | `base` / `micro_視線停止` / `micro_口元硬化` / `fatigue薄め` | Shot 1, 2, 4, 7 |
| クマ課長 | `pressure_ため息` / `pressure_無言支配` | Shot 3, 6 |
| 相沢律子 | `base` / `reinforce_無言同調` / `reinforce_視線落とし` | Shot 1, 5 |

## 音の置き方

EP001試験版では、派手なBGMより**職場の環境音とため息SE**が中心になる。もしBGMを入れるなら、存在感は低く、緊張感だけを薄く支えるものにとどめる。

| 音要素 | 入れ方 |
|---|---|
| 空調・タイピング | 全体に薄く敷く |
| ため息 | Shot 3で明確に前へ出す |
| 小さな無音 | Shot 4とShot 7の前後で有効 |
| BGM | 必須ではない。入れるなら低主張 |

## 試験版を見た後の判定メモ

試験編集版を1本作った後は、完成・不完成で判断するのではなく、以下の3点だけを確認すれば十分である。

| 判定項目 | Yesなら維持 | Noなら調整 |
|---|---|---|
| ロゴが気になりすぎない | 現サイズ維持 | サイズ縮小 or 不透明度低下 |
| 字幕が少なくても意味が通る | 現状維持 | Shot 2か4に短い補助字幕を追加 |
| 命名締めが気持ちよく落ちる | 現状維持 | 命名前の間 or 表示秒数を再調整 |

## 結論

EP001試験編集版は、**共通テンプレが本当に機能するかを測るための最初の検証回**として非常に適している。今回必要な背景画像はあるが、量は多くない。最小構成で進めるなら、**朝の執務室全景、守屋守席、クマ課長席、相沢律子席**の4背景があれば十分である。

この指示書に沿って1本作れば、次に確認すべき調整点は、ほぼ**ロゴの主張量・字幕量・命名締めの秒感**の3つに絞れる。
