# EP001 試験編集版 実制作パッケージ

## 目的

本書は、既存のEP001エピソードシート、30秒shotlist初稿、実写コンテ用コマ割り指示、共通編集テンプレートを、**そのまま編集着手に移せる実制作単位**へまとめ直したハンドオフ文書である。今回の方針は全面改稿ではなく、すでに整備済みの設計を崩さず、**実際に使うファイルパス、ショット順、仮置き判断、不足素材の最小補完方針**だけを明示することにある。

EP001試験編集版の役割は、完成納品を一気に目指すことではなく、Season 1前半の標準編集運用が成立するかを確認することである。したがって、最初の到達点は、**ロゴ常設・字幕少なめ・観測ログ型の命名締め**を備えた30秒前後の試験版を、再現可能な手順で組める状態にすることである。

## 今回の実制作で固定する前提

EP001は、朝の執務室で起きる小さな空気停止を扱う導入回であり、演出の中心はクマ課長のため息そのものではなく、**守屋守がその圧を観測してしまう瞬間**に置く。そのため、編集は速さよりも、視線停止、口元の硬直、無言同調、タイピング再開の気味悪さを読ませる構成を優先する。

| 項目 | 今回の固定値 | 実制作上の扱い |
|---|---|---|
| 尺 | 29〜31秒 | 30秒前後で固定し、命名締めを削らない |
| 画角 | 9:16 | 1080×1920 を基準にする |
| ショット数 | 7ショット | 既存shotlistを維持する |
| 主視点 | 守屋守 | 反応の読み取りを最優先にする |
| 命名型 | 観測ログ型 | 終盤で `ai_log_overlay_v01.png` を使用する |
| ロゴ運用 | 右上常設 | 画面幅12%前後、不透明度85%前後から開始 |
| 字幕方針 | 少なめ | Shot 2・4・7を中心に最小限で運用 |

## ラフ組み開始時の暫定代替方針

差分生成を待たずに編集レイアウトを先に進める場合は、既存の基準画像を暫定代替として使う。今回の目的はまず秒感、ロゴ干渉、字幕量、命名締めの体感確認であるため、**ラフ段階では表情の完全一致よりも構図と時間設計を優先**してよい。

| 不足差分 | 当面の代替 | 仮対応の考え方 |
|---|---|---|
| 守屋守 視線停止 | `moriya_mamoru_pose_bust_45_v01.png` | 斜めバストでCUを仮置きし、停止感は尺とSEで補う |
| 守屋守 薄い疲労 | `moriya_mamoru_base_face_front_v01.png` | 命名前は暗転と音の引きで疲労感を寄せる |
| クマ課長 ため息圧 | `kuma_kacho_pose_bust_45_v01.png` | 肩幅が入る構図を優先し、息SEで圧を成立させる |
| 相沢律子 無言同調 | `aizawa_ritsuko_pose_bust_45_v01.png` | 視線落としの演技はトリミングと速度調整で仮補完する |

ただし、この暫定代替は**ラフ確認用に限定**する。本番確認版へ進む時点では、上記の表情差分を順次差し替えることを前提とする。

## 字幕と命名テキストの実制作基準

字幕は、気持ちを説明するためではなく、理解を最小限固定するために使う。EP001では沈黙が機能するため、字幕を増やすほど怖さが弱まる。したがって、今回の編集着手段階では、**字幕ゼロで成立するショットは無理に埋めない**ことを標準とする。

| 区分 | 実制作基準 |
|---|---|
| Shot 1 | 字幕なしを基本とする |
| Shot 2 | 必要なら導入補助を1行だけ入れる |
| Shot 3 | 字幕なし。ため息を画と音で見せる |
| Shot 4 | 必要なら守屋守側のごく短い観測補助を1行まで |
| Shot 5 | 字幕なし |
| Shot 6 | 字幕なし、または極小の補助のみ |
| Shot 7 | 「これは、ため息ウイルスである。」を命名テキストとして使用 |

命名締めは、説明のオチではなく観測の結論に見える必要がある。そのため、Shot 7 では守屋守の停止感を一瞬残した後に暗転し、`ai_log_overlay_v01.png` を被せて命名表示へ移る。

## レイヤー実装の推奨順

実制作では、素材を都度感覚で重ねるより、レイヤーとコンポジションを固定した方が差し替えが早い。EP001では特に、ロゴと命名UIの干渉管理が重要であるため、背景と人物を先に決め、UIは最後に締める順番が安定する。

| レイヤー | 内容 | EP001での使い方 |
|---|---|---|
| L1 | 背景 | 4背景をショットごとに差し替える |
| L2 | 人物ベース | 3名の基準素材を配置する |
| L3 | 人物差分 | 守屋守停止、クマ課長ため息、相沢無言同調を差し替える |
| L4 | 小物・環境補助 | 資料、PC、机上感を必要最小限で補う |
| L5 | 字幕 | Shot 2, 4, 7 を中心に最小限運用 |
| L6 | ロゴ | 右上固定、画面幅12%前後 |
| L7 | 命名演出 | Shot 7 のみ `ai_log_overlay_v01.png` を使用 |

## 着手順序

今回のEP001は、完成映像を一発で作り切るより、**ラフ編集を短時間で通し、その後に差分と細部を詰める**進め方が最も効率的である。背景とUIは揃っているため、まずは時間設計と視線誘導を確認し、その後に人物差分を当て込む順番で問題ない。

| 手順 | やること | 到達目標 |
|---|---|---|
| 1 | 背景4枚、ロゴ、ai_log を読み込む | EP001の全ショットを配置可能にする |
| 2 | 既存の基準人物素材で7ショットを仮組みする | 30秒前後の流れを先に可視化する |
| 3 | ため息SE、環境音、命名前の薄い無音を置く | 空気停止の体感を先に作る |
| 4 | Shot 2, 4, 7 のみ字幕候補を入れる | 字幕量の過不足を確認する |
| 5 | 不足差分6点を優先順に差し替える | ラフ版を本番確認版へ近づける |
| 6 | ロゴサイズと命名秒感を微調整する | 試験版の判定対象を固定する |

## 現時点のブロッカー整理

EP001実制作の移行において、**致命的なブロッカーはない**。背景、ロゴ、命名UIは揃っているため、ラフ編集までは即着手できる。一方で、本番確認版へ進むためには人物差分6点の整備が必要であり、ここが現在の主な制作タスクである。

## 結論

EP001試験編集版は、**実制作へ移行可能な状態**に入った。今回の整備で重要なのは、新しい企画書を増やすことではなく、既存の設計を崩さず、すぐに編集へ入れる判断材料を一本化することである。本書を基準にすれば、まずは背景・ロゴ・命名UI・基準人物素材でラフ版を組み、その後に守屋守、クマ課長、相沢律子の最小差分を追加する流れで、無理なく本番確認版まで進められる。

次の実務アクションは、**EP001ラフ編集の組み立て開始**、または**不足している人物差分6点の制作着手**のいずれかである。試験版の目的がロゴ、字幕、命名締めの体感確認である以上、まずはラフ版を先に通してから差分を詰める進め方を推奨する。
