# Season 1前半 EP001〜EP005｜30秒版 shotlist 初稿

## 目的
本書は、EP001〜EP005を**30秒の実写コンテ入力単位**へ落とし込むための初稿である。今回の主目的は、各話をそのまま映像制作へ渡すことよりも、**守屋守・クマ課長・相沢律子のどの表情差分が必要か**を、ショット単位で洗い出すことにある。

## 記述方針
各話は約30秒を前提に、6〜7ショット程度へ圧縮した。1ショットごとに、画の内容だけでなく、**誰のどの顔差分が必要か**を明示している。表情差分は、image2や補助生成で先に確保すべきものを見極めるため、できるだけ**再利用可能な粒度**で記述した。

## 表情差分の見方

| 記号 | 意味 |
|---|---|
| M | 守屋守 |
| K | クマ課長 |
| A | 相沢律子 |
| base | 平常顔。基準立ち絵に近い無表情〜通常業務顔 |
| micro | ごく小さな違和感、視線停止、口元の硬直などの微差分 |
| pressure | 圧を出す顔。怒鳴らないが空気を止める顔 |
| reinforce | 静かに同調・補強する顔。感じは良いが少し怖い |
| fatigue | 消耗、諦め、疲れのにじみ |

## 優先生成順の提案
実写コンテ化と表情差分洗い出しの効率を考えると、まずは**基準顔3種 + 守屋守の主要4差分 + クマ課長の主要3差分 + 相沢律子の主要4差分**を先に押さえるのがよい。

| 優先 | 先に確保する差分 | 理由 |
|---|---|---|
| 1 | 3人の base | 全話の基準になる |
| 2 | 守屋守: 視線停止 / 目泳ぎ / 足止まり / 無表情疲れ | 5話の違いが最も出る |
| 3 | クマ課長: ため息圧 / 静かな差し戻し / 軽口指示 | 圧の種類の差分になる |
| 4 | 相沢律子: 無言同調 / 柔らかい正当化 / 理性的補強 / 依頼微笑 | 前半5話で役割の幅が見える |
