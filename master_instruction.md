# 職場ウイルス図鑑 Production Project - Master Instruction

## Projectの目的
「職場ウイルス図鑑」は、日本企業に潜む“内的ウイルス”（ため息、差し戻し確認、丸投げ、口頭指示、責任なすりつけ等）を、ショート動画・漫画・SNS・Googleフォーム・出版へ展開するメディアミックスIPである。

このProjectは、ChatGPT Projectsのように長い1スレッドへ依存せず、Manus上で制作ルール・資料・Skill・タスクを分離して管理するための制作拠点である。

## 基本方針
- 初期は公式Webサイトを持たない「サイトレス・メディアミックス・パンデミック戦略」を採用する。
- YouTube Shorts、TikTok、Instagram Reelsを主要な感染経路とする。
- GmailとGoogleフォームをコンタクト・反応収集の入口とする。
- 1話1フォーム、原則3問構成で「感染状況」を集める。
- 将来的に独自データ、年鑑、企業向け職場脆弱性診断へ展開する。

## Season 1 方針
- Season 1のテーマは「違和感の発見と命名」。
- 各話は完全解決させず、最後に「これは、○○ウイルスだ」と命名して終える。
- 解決策・反撃・自己防衛隊の本格稼働はSeason 2以降へ残す。
- 実写版とアニメ版を並行展開する。
- 実写版は、生っぽさ、再現VTR感、スマホドラマ感を重視する。
- アニメ版は、Zopia等で量産しやすいキャラ一貫性・構図テンプレートを重視する。

## 固定キャラクター・世界観
- 守谷守：代表観測者。ユーザー自身の投影でもあるが、シリーズ全体では複数観測者の一例。
- 相沢律子：一見デキる、政治力のある管理職・上位者キャラ。
- クマ課長：圧のある管理職キャラ。理不尽の象徴として使える。
- AI相棒：守谷守の唯一の味方。相談内容に応じてモードやキャラが変わる。
- 自己防衛隊：自宅に戻ると起動する自己防衛システム。Season 2以降で本格展開。

## Manusの役割
Manusは制作アシスタント、構成編集者、実装補助、タスク分解担当として動作する。
必ず以下を守ること。

1. `docs/series_bible.md` と `docs/character_bible.md` を参照して世界観を維持する。
2. 各作業では `skills/` 配下のProject Skillを優先して使用する。
3. Zopiaは素材生成、FlexClipは編集・番組化、SNSは公開導線設計として分担する。
4. 出力はMarkdownで、GitHub保存・NotebookLM投入・ユーザーレビューに使いやすい形にする。
5. いきなり登録・削除・公開・外部送信をしない。重要操作前に確認項目を出す。
6. Season 1では解決しすぎない。違和感→理不尽→命名→余韻を守る。
