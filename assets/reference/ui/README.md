# UI素材リファレンス

このディレクトリは、『職場ウイルス図鑑』のショート動画制作で使用する**UI系素材**を集約して管理するための基準フォルダです。キャラクター素材、背景素材、ブランド素材とは分けて扱い、編集工程で再利用しやすいように、用途ごとに配置先を明確にします。

Season 1前半では、命名締めや情報提示の演出で使用するオーバーレイ系素材の整備を優先します。特に `ai_log_overlay_v01.png` は、EP001、EP002、EP004、EP005 の命名締め演出で使う想定の主要UI素材として扱います。また、EP003で使用予定の `registry_card_v01.png` はカード系素材として管理します。

| フォルダ | 用途 | 想定する主な素材 |
|---|---|---|
| `overlays/` | 画面に重ねて表示する透過UI素材を格納します。 | `ai_log_overlay_v01.png`、警告枠、解析ライン、フレーム演出 |
| `cards/` | 情報カード、登録票、診断票などのカード型UI素材を格納します。 | `registry_card_v01.png`、ウイルス登録カード、要約カード |
| `chat/` | 会話ログ、メッセージ表示、通知吹き出しなどのチャット系UI素材を格納します。 | チャット吹き出し、通知帯、会話ログパネル |

命名は、既存のアセット管理方針に合わせて**英小文字スネークケース**を用い、原則として `asset_id_category_variant_v01.png` の形式に統一します。たとえば、AIログ用オーバーレイは `ai_log_overlay_v01.png`、登録カードは `registry_card_v01.png` とします。版管理が必要になった場合のみ、末尾のバージョン番号を `v02`、`v03` のように更新してください。

また、UI素材は編集時に視認性を保ちつつ、画面を圧迫しないことを前提に設計・選定します。そのため、要素の余白は過度に広げず、ショート動画の限られた画面内で情報が読み取れる**コンパクトな表示**を基本方針とします。

| 管理項目 | 方針 |
|---|---|
| 画像形式 | 原則として透過対応のPNGを使用します。 |
| 命名規則 | 英小文字スネークケース、末尾に版番号を付与します。 |
| バージョン管理 | 初版は `v01` とし、差し替え時のみ更新します。 |
| 配置方針 | 用途別に `overlays/`、`cards/`、`chat/` へ分離します。 |
| 運用方針 | 全面改稿ではなく、既存演出に対する最小修正で再利用可能な形を優先します。 |

今後、UI素材が増えた場合も、このREADMEを基準として命名・配置・差し替えを行ってください。
