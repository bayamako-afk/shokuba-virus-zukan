# Brand Asset Placement Guide

## Purpose
This directory stores lightweight brand reference assets that are reused across the project, such as the **職場ウイルス図鑑** logo.

## Recommended Placement
Place the approved logo file in this directory.

| Asset Type | Recommended Path |
|---|---|
| Main logo | `assets/reference/brand/shokuba_virus_zukan_logo_main_v01.png` |
| Alternate logo | `assets/reference/brand/shokuba_virus_zukan_logo_alt_v01.png` |
| Transparent logo | `assets/reference/brand/shokuba_virus_zukan_logo_transparent_v01.png` |

## Naming Rule
Use the following convention.

> `shokuba_virus_zukan_logo_{variant}_v01.{ext}`

Examples include `main`, `alt`, `transparent`, `square`, and `mono` when needed.

## Operating Note
Keep only approved and reusable logo assets here. Do not store large batches of experiments. If a later approved version replaces an earlier one, increment the version suffix such as `v02`.
