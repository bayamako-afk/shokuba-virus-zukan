# SNS Publish Skill

このスキルは、「職場ウイルス図鑑」の動画を各SNSプラットフォームに投稿するためのテキスト（タイトル、説明文、ハッシュタグ）を作成するガイドラインです。

## 1. 投稿文の基本ルール
*   **サイトレス・パンデミック戦略の実行**: 各媒体を独立した「感染経路」として扱い、相互の導線（Googleフォーム、Kindle等へのリンク）を必ず設置します。
*   **共感の喚起**: 視聴者が「うちの職場にもいる！」と共感できるような問いかけをタイトルや冒頭に入れます。
*   **余韻の重視**: 動画の結末（○○ウイルスの命名）に触れつつ、解決策は書かずに視聴者のコメントを促します。

## 2. プラットフォーム別の最適化
### YouTube Shorts
*   **タイトル**: 検索されやすいキーワード（「あるある」「ブラック企業」「上司」など）を含め、クリックしたくなるフックを作ります。
*   **説明文**: 概要欄に、Googleフォーム（感染度チェック）やKindle（感染速報版）へのリンクを明確に記載します。
*   **ハッシュタグ**: 関連性の高いタグを3〜5個程度付けます。

### TikTok
*   **キャプション**: 若年層向けに、短くインパクトのある文章にします。絵文字を適度に使用します。
*   **導線**: プロフィールリンク（Linktree等）への遷移を促す文言（「プロフのリンクから感染度チェック！」など）を入れます。
*   **ハッシュタグ**: トレンドのタグや、エピソード固有のキーワードを多めに付けます。

### Instagram Reels
*   **キャプション**: 視覚的な訴求を補完する、少し長めの文章でも読まれる傾向があります。改行を上手く使い、読みやすくします。
*   **導線**: プロフィールリンクへの誘導に加え、ストーリーズでの共有を促す文言を入れます。
*   **ハッシュタグ**: 幅広い層にリーチするため、多数のタグ（最大30個まで）を効果的に使用します。

## 3. 出力フォーマット
以下のフォーマットで各プラットフォーム向けの投稿文を出力してください。

```markdown
# SNS Publish Draft: EP[番号]

## 共通設定
*   **対象エピソード**: [ウイルス名]
*   **フォームURL**: [Googleフォームのリンク]
*   **Kindle URL**: [Kindle感染速報版のリンク]

## YouTube Shorts
*   **タイトル**: [検索キーワードを含むフックのあるタイトル]
*   **説明文**: [動画の簡単な紹介と、フォーム/Kindleへのリンク]
*   **ハッシュタグ**: #[職場あるある] #[ブラック企業] #[エピソード固有キーワード]

## TikTok
*   **キャプション**: [短くインパクトのある文章、プロフへの誘導]
*   **ハッシュタグ**: #[トレンドタグ] #[職場あるある] #[エピソード固有キーワード]

## Instagram Reels
*   **キャプション**: [読みやすい文章、プロフ/ストーリーズへの誘導]
*   **ハッシュタグ**: #[多数の関連タグ]
```
