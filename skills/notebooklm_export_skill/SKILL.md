# NotebookLM Export Skill

このスキルは、「職場ウイルス図鑑」プロジェクトで蓄積されたデータ（エピソード情報、キャラクター設定、SNS投稿文、フォーム設問など）を、NotebookLMなどの外部ツール（LLM）で活用しやすい形式に出力するガイドラインです。

## 1. エクスポートの目的
外部ツール（NotebookLM等）を利用し、既存のデータセットを学習させることで、新たなエピソードのアイデア出し、キャラクターの深掘り、SNS運用戦略の分析などを効率化します。

## 2. データ抽出の基本ルール
*   **構造化された出力**: Markdown形式で、見出しやリストを用いてデータを整理し、LLMがパースしやすいようにします。
*   **関連情報の集約**: エピソードごとに、脚本、Zopiaプロンプト、SNS投稿文、フォーム設問などの関連情報をひとまとめにして出力します。
*   **不要情報の除外**: LLMの学習に不要なシステム的なメタデータや、一時的な作業メモは除外します。
*   **個人情報の保護**: 出力データに個人を特定できる情報（フォームの回答者情報など）が含まれていないことを確認します。

## 3. 出力フォーマット
以下のフォーマットでエクスポートデータを出力してください。

```markdown
# NotebookLM Export Data: 職場ウイルス図鑑

## 全体設定
*   **プロジェクト概要**: [プロジェクトの目的と基本方針]
*   **ターゲット層**: [ターゲット層の定義]
*   **メディアミックス戦略**: [パンデミック戦略の概要]

## キャラクター情報
*   **[キャラクター名]**: [年齢、役職、特徴]
*   ...

## エピソードデータ

### EP[番号]: [ウイルス名]
*   **テーマ・違和感**: [テーマ]
*   **登場人物**: [キャラクター名]
*   **脚本要約**: [脚本の簡単な要約（フック〜結末）]
*   **Zopiaプロンプト例**: [代表的なプロンプト]
*   **SNS投稿文（YouTube Shorts）**: [タイトル、説明文、ハッシュタグ]
*   **フォーム設問**: [設問1〜3の要約]

### EP[番号]: [ウイルス名]
...
```
