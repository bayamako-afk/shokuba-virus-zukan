# Episode Script Skill

## 目的
Season 1用の15〜30秒ショート動画脚本を作成する。構成は4〜6カットを基本とする。

## 固定ルール
- 守谷守を代表観測者として扱う。
- 相沢律子、クマ課長はテーマに応じて使い分ける。
- Season 1では完全解決しない。
- 最後に「これは、○○ウイルスだ」と命名する。
- 視聴者が「うちにもある」と思えるリアルな職場あるあるにする。

## 出力形式
```markdown
# EP[番号]｜[ウイルス名]

## ねらい
## 登場人物
## 15〜30秒版カット割り
| Time | Scene | Dialogue/Voice | Caption |
|---|---|---|---|
## ラスト命名テロップ
## Zopia素材候補
```
