# Form Question Skill

このスキルは、「職場ウイルス図鑑」の各エピソードに連動したGoogleフォーム（感染度チェック）の設問を作成するガイドラインです。

## 1. フォームの目的
Googleフォームは、視聴者の「感染度」を測り、コンテンツへの没入感を高めるとともに、将来的なデータ活用（年鑑化、企業向け診断）の基盤となります。

## 2. 設問作成の基本ルール
*   **1話1フォーム、原則3問構成**: ユーザーの回答ハードルを下げるため、設問数は3問に絞ります。
*   **シンプルで答えやすい選択式**: 動画で描かれたウイルスの特徴（症状）に合致する行動や状況を問う、選択式の設問（「よくある」「たまにある」「ない」など）を中心に構成します。
*   **個人情報の非取得**: 初期段階では、氏名やメールアドレスなどの個人情報は取得しません。
*   **完了画面の最適化**: フォーム送信後の完了画面には、関連動画やKindle「感染速報版」、SNSへのリンクを必ず設置し、回遊を促します。

## 3. 出力フォーマット
以下のフォーマットでGoogleフォーム用の設問リストを出力してください。

```markdown
# Google Form Questions: EP[番号]

## フォーム基本情報
*   **タイトル**: [ウイルス名] 感染度チェック
*   **説明**: あなたの職場に「[ウイルス名]」は潜んでいませんか？3つの質問で感染度をチェックしましょう！（※個人情報は取得しません）

## 設問リスト

### 設問1
*   **質問文**: [ウイルスの特徴を示す具体的な行動や状況について問う質問]
*   **形式**: ラジオボタン（単一選択）
*   **選択肢**:
    *   よくある（または「はい」）
    *   たまにある（または「どちらとも言えない」）
    *   ない（または「いいえ」）

### 設問2
*   **質問文**: [ウイルスの別の特徴を示す質問]
*   **形式**: ラジオボタン（単一選択）
*   **選択肢**: [設問1と同様]

### 設問3
*   **質問文**: [自身がウイルスキャリアになっていないかを問う、少しドキッとする質問]
*   **形式**: ラジオボタン（単一選択）
*   **選択肢**: [設問1と同様]

## 完了画面メッセージ
診断完了！あなたの感染度は…？
結果はSNSでシェアして、職場の仲間にもチェックしてもらいましょう！
👇[ウイルス名]の詳しい解説はKindle「感染速報版」で！
[Kindleのリンク]
👇他のウイルスもチェック！
[YouTube/TikTokのリンク]
```
