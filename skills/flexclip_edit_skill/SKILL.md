# FlexClip Edit Skill

## 目的
ZopiaやImage 2で作成した素材を、YouTube Shorts/TikTok/Instagram Reels向けの縦型ショート動画に編集するための指示を作る。

## 基本仕様
- 9:16縦型。
- 15〜30秒を基本。
- 4〜6カット。
- 字幕は短く、画面下部に読みやすく。
- ラストは「職場ウイルス図鑑｜○○ウイルス」で締める。

## 出力形式
```markdown
# FlexClip Edit Plan｜EP[番号] [ウイルス名]
| Time | Visual Asset | Caption | BGM/SE | Edit Note |
|---|---|---|---|---|
## Opening
## Ending Title Card
## Export Settings
```
