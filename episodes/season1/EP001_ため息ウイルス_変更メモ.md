# EP001「ため息ウイルス」変更メモ

今回の修正は、**全面改稿ではなく最小修正**に限定した。Season 1 第1話としての骨格、30秒尺の前提、そして「大事件は起きていないのに空気だけが壊れる」という核は維持している。

## 主な修正点

| 項目 | 修正内容 |
|---|---|
| 名前の統一 | `守谷守` を **守屋守** に統一した。 |
| 相手側の具体化 | 抽象的な「課長」「上司圧力型」を、この回では **クマ課長** として明示した。 |
| 相沢律子の整理 | 相沢律子は直接の加害源ではなく、**無言で空気に同調し緊張を補強する存在**として整理した。 |
| 主観カット強化 | 守屋守の主観として、**キーボード上で指が止まり、課長席へ視線が流れ、その先で周囲の肩がわずかに縮こまる**描写を加えた。 |
| キャラ解釈の調整 | 守屋守、相沢律子、クマ課長の年齢感・雰囲気・圧の出し方を、共有された実写基準の印象に寄せて整えた。 |
| 演出メモ補強 | 主観カットの軸を明記し、守屋守の観測性が映像で伝わるようにした。 |

## 維持した点

| 項目 | 維持内容 |
|---|---|
| 話の骨格 | 朝のオフィス、ため息、空気停止、観測、命名、暗転という流れは維持。 |
| 尺設計 | 30秒主力、15秒・45秒の派生構造は維持。 |
| トーン | 社会怪談的な不気味さと、説明しすぎない演出を維持。 |
| Season 1方針 | 反撃には入らず、**違和感の発見と命名**で止める構成を維持。 |
